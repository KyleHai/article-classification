期货日报网讯日，由郑州商品交易所、延长县人民政府主办，长江期货股份有限公司、延长县果业中心、延安中果生态农业科技有限公司承办的“稳企安农护航实体——苹果期货助力脱贫攻坚（延长）论坛”在陕西省延长县召开，郑商所、长江期货、人民银行、农业银行、延长县信用联社、人保财险等企业负责人及延长县果业局、苹果种植大户等130余人参加培训会	0
论坛主要围绕苹果期货基础知识、苹果产业市场形势分析、苹果期货交割实操、苹果“保险+期货”介绍及案例分享等内容进行了系统的讲解	0
2020年延长县苹果“保险+期货”项目是今年全国三个苹果全覆盖“保险+期货”项目之一	0
该项目承保苹果种植面积2万余亩，覆盖产量3万余吨	0
延长县“苹果+期货”投保时间是9月10日至9月16日，仅一周时间就有1000余户果农积极参与投保，其中还包括近300户贫困户	0
苹果产业是延长县兴县富民的主导产业，也是脱贫攻坚的核心切入点，以苹果为主的绿色产业已经成为延长县果农增收致富的支柱产业	0
近年来，为推动延长县苹果产业积极利用苹果期货、苹果“保险+期货”等新型金融工具，长江期货联合郑州商品交易所、延长县政府，通过加强苹果专业合作社、果农大户等市场培育，积极开展“保险+期货”试点，推动延长苹果产业高质量发展，以实际行动助力延长县脱贫攻坚	0
苹果“保险+期货”模式可以充分发挥期货和保险行业专业化、精准化、特色化、创新化等“专精特新”优势，提振群众发展产业的信心，拓宽群众稳定增收渠道，为苹果产业的发展带来新助力，进一步打开苹果产业升级空间	0
