<pclass='ori_titlesource'>原标题：2020年10月13日，和讯期货：富县2020年苹果“保险+期货”试点工作启动暨政策业务培训会来源：新闻信息部易盛	0
近日，富县2020年苹果“保险+期货”试点工作启动暨政策业务培训会在富县人民政府会议中心召开，富县县委常委马洁主持会议，副县长许新选、富县果业局局长王延林、郑州商品交易所、太平洋财产保险陕西分公司、方正中期期货、浙商期货、国投安信期货及富县相关乡镇负责人和果农代表出席了此次会议	0
会上，富县果业局王延林局长宣读了富县2020年苹果“保险+期货”试点工作实施方案，并就全县8个苹果产业乡镇开展苹果“保险+期货”试点工作任务及重点事项进行了安排部署	0
2020年富县项目采取“贫困户优先，一般户自愿”的原则，富县8个苹果主产乡镇从事苹果种植的建档立卡贫困户可优先投保，争取做到贫困户2000余户全覆盖，预计参保面积将达到2.3万亩，可保障苹果现货约3.46万吨，其中，方正中期期货约保障苹果现货8600余吨	0
富县副县长许新选做总结讲话，要求各乡镇提高政治站位，高度重视项目工作，为富县苹果产业持续健康发展，夺取富县脱贫攻坚战的胜利打好基础	0
未来，方正中期期货将继续在富县深化开展精准扶贫工作，建立扶贫长效机制，助力富县苹果产业结构优化升级、健康发展	0
