苹果：霜降即将来临产区价格短期见顶回落	0
<pclass='ori_titlesource'>原标题：苹果：霜降即将来临产区价格短期见顶回落来源：混沌天成研究	0
国庆假期之后，苹果强势拉涨，山东开称高开高走（高开指的是较预期高，实际和去年持平），山东苹果上市之后，优果率下降成为市场关注的重点，此次调研主要是为了探究山东今年优果率问题，以及价格走高的原因，从而辅助对后市价格走势做出预判	0
据国家统计局统计，2019年全国苹果产量4243万吨（卓创资讯的口径是4266万吨），其中山东产量872万吨，占全国产量20.5%	0
据前期各调研机构的调研数据显示，因西部今年花期受冻减产，而前期预估山东增产15-20%，所以综合预估20-21产季全国产量同比-5%左右，而随着山东苹果开始上市，今年因气候、人为留果较多等原因，山东产区增产不及预期，预估增产幅度下降在5-10%，此预期差将近有80万吨，约影响全国1.5-2%的产量	0
而反观下游消费，从宏观数据方面，我们可以看到社会消费品零售总额仍未恢复到去年水平，且分项来看，今年疫情对于农村的消费打击更大，主要是低收入人群的消费能力下降了；另一方面，从农业农村部批发市场富士苹果的周度交易量来看，4月份以来，消费持续低迷，今年的国庆节日效应也未能带动走货加快，说明终端消费未恢复到正常年份	0
