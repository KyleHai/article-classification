天水花牛苹果喜迎丰收季78吨苹果首发	0
<pclass='ori_titlesource'>原标题：天水花牛苹果喜迎丰收季78吨苹果首发	0
日前，全省苹果产业现场推进会在天水市麦积区举办，天水市麦积区金融支持苹果收储营销专项贷款发放仪式及天水花牛苹果发车仪式同期举行	0
8辆货车满载78吨花牛苹果从天水花牛苹果产业园缓缓驶出，发往北京、浙江嘉兴、广州东莞、云南昆明等地，为今年天水花牛苹果销售夺取“开门红”	0
麦积区先后建成城郊南山、中滩九龙山、石佛北山等万亩以上集中连片优质花牛苹果基地11个，创建国家级、省级标准化示范园24处8.2万亩，发展千亩以上苹果种植村145个，全区苹果种植面积达42万亩，预计今年总产量70万吨	0
为给花牛苹果收储营销提供扎实资金保障，麦积区在去年完成发放5.36亿元苹果收储贷款的基础上，今年积极筹措1000万元风险保证金，在全市率先启动实施了中国人保1亿元“险资直投”政融保项目和3000万元融兴村镇银行果品收储专项贷款	0
发放仪式上，相关金融机构代表向天水维亚实业有限责任公司、天水森源农林科技有限公司、天水昊源农艺有限公司、天水洁通农业科技有限公司、麦积区成功农民种植专业合作社等5家果品营销企业发放1亿元中国人保“险资直投”项目融资贷款资金；向麦积区农业产业龙头企业协会发放3000万元苹果收储贷款；向麦积区扶贫公司发放2500万元苹果收储贷款；向天水绿之源农业发展有限公司发放300万元苹果收储贷款	0
麦积区多方衔接协调各类金融机构积极开发果品收储信贷产品，组织实施苹果“保险+期货”项目，目前已发放各类果品收储贷款4.32亿元	0
另外，为持续巩固脱贫攻坚成果，麦积区扶贫公司筹措资金3000万元，联合麦积区内多家果品营销企业在苹果主产区设立集中收购点，以贫困户为重点进行保底价收购，确保果农的新果及时收储、顺利销售，卖出好价钱	0
