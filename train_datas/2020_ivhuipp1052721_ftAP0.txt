永安期货王毅：新季苹果的展望	0
纵观年后苹果市场走势，可以说让市场即是意料之外又在意料之中	0
让市场意料之外的是现货价格能下跌的如此顺畅，意料之中的是今年年后超历史的库存水平叠加国内因为疫情导致的消费下滑，使得苹果的现货价格从5月份一路跌到8月	0
当下进入8月的尾声，随着新季苹果的陆续上市和冷库老果的库存下滑，市场又到了一个比较胶着的时刻，在这里让我们重新回顾历史并且展望新年度苹果市场	0
从年后的现货价格走势来看剔除4月份的小幅反弹外，今年山东的苹果现货价格一路下滑，尤其是进入6月份以后下滑的斜率开始加大，主要还是根据我们之前的判断，认为在今年大库存的背景下叠加疫情的影响使得国内苹果消费能力大幅下滑，这也迫使苹果贸易商和冷库不得不采取以价换量的模式来销售	0
类同于15/16年的大库存弱消费格局，不同的是冷库与贸易商们错过了最佳销售时期，该降价刺激市场消费的时候反而挺价捂货不出，最后错过了3-5月这段销货黄金期，导致到了5月份苹果冷库库存依旧巨大，最后由于销货时间的减少加上库存的压力造成了大家挤仓出库使得苹果价格瞬间崩塌	0
从历史的经验看来，在消费不好的年景去苹果库存的唯一方式只有降价，降价对库存去化效果明显	0
从今年7月份后的行情可以看出，当市场苹果价格快速下滑之后的确刺激了市场的消费使得库存下降幅度加快，截止到8月20日，全国库存已经下降到103.8万吨，山东库存下降到77.3万吨，不过由于前期浪费的时间过多，导致目前库存依旧处于历史高位水平尤其是山东产区	0
首先因为收入的下滑导致非必要消费的缩减，苹果作为一个非必要消费品需求下降也在预期之中；其次，市场经济不景气使得大量企业难以正常展开工作造成员工大面积流失，例如纺织服装、旅游、出口加工厂和航空等行业都出现了裁员现象，这一现象也使得这部分人员的消费水平大幅下降；最后，也是因为肺炎疫情的影响导致今年国内旅游人数的下降，人们为了避免聚集性的事件发生取消了旅行计划使得原先旅行中产生的水果消费大幅度的减少了	0
从大方向来看今年新季苹果整体格局还是整体偏弱	0
供应端无论是从前期的调研情况或现阶段早熟摘袋的情况来看，今年苹果产量较之前正常年份没有明显的下滑，产量属于正常年份水平；需求端目前来看虽然国内肺炎疫情基本已经结束，但是疫情对国内经济与消费的影响还将延续，从前期的时令水果的冲击和当下早熟苹果的高开低走行情看至少到今年年底市场都没有明显消费好转的迹象	0
综上所述，在整体格局偏弱加上同期消费下滑的背景下，我们预计今年新季苹果开称价格将弱于往年，晚熟富士开秤价格处于2.6元/斤左右，目前盘面价格还有一定升水整体价格还有向下的空间	0
