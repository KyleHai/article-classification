<pclass='ori_titlesource'>原标题：中国农科院果树研究所苹果试验站正式获批落户栖霞	0
中国苹果看烟台，烟台苹果看栖霞	0
作为烟台苹果核心产区的栖霞市，自上个世纪80年代以来，一直引领中国苹果产业的发展，先后获得“中国苹果之都”“中国水果第一市”等荣誉称号	0
为全面贯彻省果业发展现场会议精神，落实《山东省推动苹果产业高质量发展行动计划》，推动烟台(栖霞)苹果产业再造新优势、再攀新高峰，栖霞市抢抓机遇，不断加大“双招双引”工作力度，特别是在引进建设“国字号”科研机构上下功夫、求突破，“中国农科院果树研究所栖霞苹果试验站”应时而生	0
通过试验站，栖霞市引进高层次果业专家22名，引进国内外苹果新优品种56个，定制“十大主推技术”，开展“百名博士进栖霞”活动，开展线上技术培训讲座，围绕推动苹果产业高质量发展的一系列举措正徐徐展开	0
科技联姻，构筑苹果高质量发展新引擎	0
为切实推动苹果产业转型升级、提质增效，栖霞市把科技创新作为引领苹果产业高质量发展的第一动力，通过外招内引、联合共建的方式，吸引“国字号”科研机构入驻栖霞，为苹果高质量发展注入不竭动力	0
该市成功与中国农科院果树研究所联合举办“中国苹果产业高质量发展会议”，签订共建“中国农科院果树研究所栖霞苹果试验站”合作协议，这是该市首次引进“国字号”院所在本地建设的科研机构，也是中国农科院果研究所设在东部地区的首个苹果试验站	0
为加快推进栖霞苹果试验站落地运行，栖霞市政府主要领导亲自部署、亲自调度，主持召开市政府常务会议专题研究建站事宜，将试验站建设纳入全市“三重”工作，定期督导落实	0
2020年3月，中国农科院果树研究所栖霞苹果试验站管理处正式成立，为正处级科研机构，作为中国农科院果树研究所的外驻部门，由双方共同选派人员进站开展工作	0
科学规划，打造苹果科技创新新高地	0
栖霞苹果试验站规划总面积420亩，主要包括综合科研实验区20亩、中试示范区300亩、技术集成示范区100亩	0
中试示范区主要建设品种选育区、苗木繁育区、设施果业试验区，配套连栋现代智能玻璃温室和气象观测站等，是苹果试验站的试验示范基地	0
技术集成示范区主要将试验站研发的先进技术成果进行集成应用，是苹果试验站的成果展示基地	0
栖霞苹果试验站重点开展苹果产业新品种、新技术、新成果研发与试验示范，打造苹果产业的品种培育基地、成果转化基地、技术示范基地、人才培养基地、国际合作基地和科普教育基地，是推动苹果产业高质量发展的重要创新载体和智库	0
通过建设栖霞苹果试验站，将极大提升栖霞乃至烟台苹果的科技含量和科技水平，加速苹果产业迈入智慧化、高端化发展轨道，为推动农业现代化和乡村振兴作出新的更大贡献	0
围绕推动苹果产业绿色高质量发展，打造苹果产业研发创新高地和高标准试验示范基地，栖霞市充分发挥人才作用，通过苹果试验站柔性引进中国农科院果树研究所党委书记、国家苹果产业技术体系岗位专家丛佩华，中国农科院果树研究所副所长、国家苹果育种中心副主任程存刚等高层次果业专家22名，与中国农科院果树研究所研究确定苹果试验站专家团队，涵盖苹果全产业链的15个课题组，探讨开展“百名博士进栖霞”活动，采取选派博士专家结对包帮镇村、企业的形式，帮助指导解决生产过程中遇到的实际困难，示范推广先进技术、成果等，为苹果产业发展提供高层次人才和智力支撑	0
试验站专家们根据栖霞地区特色量身定制了“两无两改降成本、三精三高增效益”的“十大主推技术”，引进推广国内外苹果新优品种56个，制定《栖霞苹果试验站2020年工作任务》，明确了8项重点工作任务的实施方案、责任人和实施时间，全力保障试验站科研、示范、培训等各项工作的顺利实施	0
