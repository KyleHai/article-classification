山东苹果·果业产业技术研究院成立	0
<pclass='ori_titlesource'>原标题：山东苹果·果业产业技术研究院成立来源：记	0
7月29日上午，山东苹果·果业产业技术研究院揭牌暨签约仪式在烟台举行	0
作为全国首家苹果类专业“四不像”（既不像机关，又不像学校；既不像企业，又不像事业单位）新型研发机构，和全省唯一的农业类创新创业共同体，山东苹果·果业产业技术研究院以集聚果业科技创新资源、培育发展新兴产业、支撑传统果业转型升级为宗旨，组织开展苹果及果业产业技术研究和集成攻关，建成需求引导、多元共建、统分结合、体系开放、接轨国际、全国领先的苹果·果业创新创业共同体	0
截至目前，苹果果业创新创业共同体合作单位已有50多家	0
苹果产研院的成立为推动烟台苹果产业高质量发展按下了加速键	0
烟台市苹果种植面积280万亩，年总产量570万吨	0
全市苹果种植户约80万户，从业人员170多万人	0
烟台苹果正处于结构调整、方式转变、动能转换的关键阶段	0
据介绍，山东苹果·果业产业技术研究院将着力破除制约科技创新的思想障碍和制度藩篱，不断探索理念创新、体制创新、路径创新、项目创新、资金创新：院长通过市场公开招聘的方式选聘，全员聘任、以岗定薪、以效定酬，打造薪酬分配“特区”；构建“政产学研金服用”紧密融合的全社会协同创新体系，打造协同创新的“先行区”；推行项目经理制、重点项目攻关“揭榜挂帅”制，释放人才要素活力，打造创新创业的“试验区”；组建苹果产研院投资公司，引进外部资源，联合金融机构，设立股权多样化的苹果产业发展基金，助推创新创业	0
山东苹果·果业产业技术研究院还将重点建设7个科研创新中心，组建8个创新团队，实施7项重点创新工程	0
同时，烟台市大数据局联合市农业农村局建设烟台苹果大数据中心，通过连线大数据中心的展示平台，可以看到全市苹果各主产区的苹果种植规模、产量、加工产品生产量等数据，实时显示烟台苹果销售价格，大数据中心预计8月底或9月初投入使用	0
