5个交易跌800元/吨霜降让苹果价格“冻得打哆嗦”	0
昨日，苹果期货主力合约的价格又跌回到了国庆节前（9月30日）的水平，如果从10月12日的最高价8412元/吨算起，至昨日的盘中最低价7508元/吨，短短十几日苹果主力合约期价累计下滑了904元/吨，而如果从霜降节气开始发挥利空作用的10月20日的最高价8308元/吨算起，期价在5个交易内累计跌幅为800元/吨	0
为什么短时间内苹果市场价格会出现这么大的波动呢	0
期货日报记者通过与市场人士沟通发现，苹果的市场情绪和预期也出现了翻天覆地的改变	0
苹果市场情况情况或如是	0
苹果期货研究者方红江告诉期货日报记者，国庆节前后与霜降节气前后苹果期现货价格的大幅波动，如果从市场基本面去解释，一时很难讲清楚，因为苹果的产量、库存量和需求量均不会在短时间内发生大的改变，因此，此次价格的大幅波动基本上是市场情绪与预期发生了较大变化而引发的	0
方红江认为，国庆节期间与国庆节后数日内苹果期现货价格的上涨，特别是陕西等地今年新果开秤价较高令市场人士惊讶不已，甚至让很多果商与果农摸不着头脑	0
他与部分专业人士沟通得出的结论是：新果价格的高开高走起初主要是受到了甘肃等地苹果产量与质量不及预期的提振	0
随后，由于有较多苹果贸易商从甘肃等地转移到陕西采购，而在陕西又收不到价格合适的苹果，苹果贸易商转而把目光转向了山东烟台以及其他苹果主产区	0
如此以来，今年新果产量减少引起的价格高开高走的市场情绪与预期得到顺利传递，进而不断带动烟台苹果价格走高，最终令国内新产苹果价格在开秤初期整体处在高位	0
“10月14日，这里的苹果刚下树时，针对80条纹果，我们公司尝试挂出了2.8元/斤的收购价，公司周边村庄果农对此比较认可，收购量不断增加，新果收购进行得很顺利	0
”山东栖霞市一位苹果收储商告诉记者：“不过，好景不长，10月16日开始有人在公司收购点附近挂出了3—3.2元/斤大量收购80条纹果的招牌，眼看着收购量直线下降，公司不得不同步抬价收购	0
”他认为今年栖霞等地苹果价格高开高走有人为推动的因素	0
10月12日至17日，记者在陕西与山东的苹果市场调研时发现，很多市场主体看好苹果价格，因为今年苹果优果率下降、陕西与甘肃等地苹果减产，再加上蒜商、姜商、梨商等入市，甚至商超果和直销果需求增长等均成为了苹果上市之初价格上涨的题材	0
但随后不久，随着山东烟台等地苹果的大量上市，市场人士很快发现大量收购并囤积苹果存在较大风险，因为市场的总走货量并不大，而且烟台苹果增产的概率也较高	0
据方红江介绍，发现苗头不对后，“老道”的苹果贸易商与冷库负责人率先开始谨慎收购，进而是果农积极出售，而一些原来较为看好后市的贸易商与大果农也改变了态度，更何况霜降节气之后，苹果不宜长时间挂在树上，因此在霜降节气来临前后，国内主要苹果产区特别烟台苹果市场价格出现了较大幅度下滑	0
据了解，霜降前后，各地苹果市场价格均出现了松动，贸易商与冷库的收购标准得到了加强，同时新果集中上市，入库量也开始放大	0
不过，在价格下滑前，陕西等地苹果已基本采摘完毕，很多苹果也已签订了出售订单，因此，烟台苹果的大量上市与价格下滑究竟会对未来市场产生多大的影响还需全面分析	0
面对当前苹果市场行情的反转，有人认为是国庆节前进场做多者获利平仓所致，毕竟在7300—7500元/吨区间进场做多者已获利丰厚；有人认为是烟台苹果集中上市、收购价突然下滑和入库量预期由不足转为“炸库”等因素引发的，毕竟烟台苹果是用于期货市场交割的主要资源；有人认为是市场需求不足与新冠肺炎疫情在青岛等地复发，以及国外疫情蔓延的结果，毕竟最终还需要靠需求才能支撑价格上涨	0
不管是哪种理由，国庆节前后市场看涨苹果价格的题材此时已消失无影踪，高位进场的投资者或及时止损或“黯然神伤”了	0
从苹果红、价格高、果农乐到霜降来、价格低、果农愁，“变化无穷”的苹果期现货价格更令广大期货投资者感到“无所适从”	0
