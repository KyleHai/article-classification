展望于2020年下半年，冷库苹果进入清库阶段，不过居高库存及品质下降，造成去库难度加大，同时应季水果大多丰产，价格偏低于往年，对苹果需求带来一定冲击	0
随着8月早中熟苹果逐渐上市，供应持续增加，苹果供需基本面仍偏弱，预计下半年苹果市场仍难言可观	0
一、2020年上半年苹果市场回顾	0
2020年上半年，苹果期货价格指数变化的主导因素为苹果冷库库存变化、产区天气及其他替代品水果等，以下做详细的解析	0
（一）苹果期货价格指数走势回顾	0
2020年上半年苹果期货价格走势细分为三阶段	0
2020年1月初-4月中旬，受疫情影响，苹果产区冷库出货、批发市场等需求备货减弱，不过替代品价格相对坚挺及下方成本支撑等多重因素作用下，苹果价格指数呈现宽幅震荡运行	0
此期间，苹果期货指数处于6500-7800元/吨区间宽幅震荡	0
2020年4月下旬-5月12日，受霜冻天气减产预期以及冰雹等不良气候提振，苹果期货价格指数低位反弹，此期间从7200元/吨上涨至高点9330元/吨，涨幅约为29.60%	0
5月底-至今，随着应季水果逐渐上市，水果市场供应相对充裕，消费者逐渐转移至价格低廉的时令鲜果，苹果终端需求进一步受到削弱，促使苹果价格指数持续下跌，此期间苹果价格指数从高点9330元/吨跌至8000元/吨附近，跌幅约为14.00%	0
