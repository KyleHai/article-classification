“谈及洛川苹果的发展历史，不得不提被誉为‘洛川苹果之父’的洛川县永乡乡阿寺村村民李新安	0
”屈军涛介绍，早在1947年，李新安从河南地区学习了几年苹果树栽培技术后，用毛驴驮回了200株苹果树苗，在自家6.7亩麦田上试种出了洛川县第一个苹果园	0
很快，李新安的果园迎来丰产期，产出的高品质的苹果，带来了优渥的经济收益，为乡亲们打开了一条致富增收的新途径	0
“实际上，洛川苹果的高品质，不是偶然，而是独特的地域条件所决定的	0
根据世界苹果名优产区气候优生资源标准，洛川是我国屈指可数的全部符合苹果生长七项气候指标要求的苹果优生区，是中国苹果优生区里的‘黄金带’	0
”屈军涛笑着说，“具体来讲，洛川县具有海拔高、光照充足，昼夜温差大，雨热同期，土层深厚四大优生地域条件，保证了洛川苹果的品质	0
“靠天吃饭”不如“知天而作”，屈军涛介绍，光有优生地域条件远远不够，洛川苹果的发展也离不开几代科研团队的长期“深耕”和全县干部群众的艰苦创业	0
目前，洛川县已建成以西北农林科技大学延安洛川苹果试验站、洛川苹果院士工作站为核心的人才技术团队，汇聚了来自国内外50余名技术专家和学者	0
同时，近年来，洛川县还大力推广苹果种植“一村一技”“一户一策”，及时地将先进的种植技术“花开”在全县群众的田间	0
有好品质，还要守得住好品牌，屈军涛介绍，近年来，洛川县委县政府全方位促进果品的质量安全建设，利用标准化建园，标准化生产，标准化质检推动苹果产业走上了高标准发展之路	0
当前，洛川县引进33条苹果品质高端智能选果线，全县53万亩果园已通过国家绿色食品原料生产示范基地的认证，家家都有可实现产地质量追溯的苹果电子“出生证”	0
如今，洛川苹果远销28个国家和地区，销售市场覆盖全国大多数一、二线城市，获得“2008年北京奥运会专供苹果”“2010年上海世博会指定苹果”等30多项品牌和冠名权，荣获省部级以上大奖180多项，并多次作为“国礼”赠送外国元首	0
