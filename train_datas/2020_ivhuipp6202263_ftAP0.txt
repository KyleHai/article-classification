苹果：市场需求疲软，价格仍存下跌可能	0
<pclass='ori_titlesource'>原标题：苹果：市场需求疲软，价格仍存下跌可能	0
征稿(作者：弘业期货周競）--2020年以来，苹果在诸多因素的综合作用下出现了下跌行情，到目前为止，并没有有效反弹	0
水果市场整体需求欠佳，导致下游消费不畅，疫情对消费的影响不言而喻:苹果、梨子、西瓜、葡萄等大众水果售价均出现下滑.此外，2020年苹果产量将再创新高，疫情导致其出口受阻，苹果“出口转内销”，进一步加剧了其市场销售压力	0
，从各地批发市场报价来看，前期早熟苹果和冷风库库存出现竞争，早熟苹果卖不上好价，冷库库存也在降价走货	0
10-11月为传统苹果收获期，大量晚熟苹果即将上市，供应增加，价格下跌概率较大	0
苹果需求疲软压力较大	0
今年在疫情影响经济收入的背景下，人们的消费能力下降，上半年春节本来是苹果消费的旺季，但今年突发疫情，导致消费和物流双双受到影响，销售节奏被打乱，苹果及其深加工产品的消费需求下滑，促使价格回落	0
苹果做为普通农产品，具有季产年销的属性，且储藏时间有限，并不具备特殊的炒作属性，唯独在供需规模发生明显大幅度改变的背景下才有望被市场炒作	0
到了下半年，苹果又面临其他替代水果的竞争：苹果不仅主产区在中国西北部，主要消费群体也集中在中国西北部地区，虽然苹果产量大，但是在南部地区，由于气候适合，可以选择种植的水果更多，意味着苹果的可替代品更多	0
南方地区主要的替代水果包括砂糖橘、香蕉，而全国范围内，苹果的替代品包括西瓜、桃子等普通大众消费水果	0
鸭梨、富士苹果价格同比分别下55.8%、46.7%；西瓜、香蕉、巨峰葡萄环比分别下跌31.9%、9.8%、4.7%	0
我国夏季消费水果中占比最高的为西瓜，然后是桃子、葡萄、苹果、香蕉	0
苹果在夏季水果消费中并不占据优势，而柑橘和梨子都是秋季上市，又对当季新苹果的消费产生竞争影响	0
客商采购不积极，库存苹果交易速度缓慢，早熟苹果质量差，陕西地区上市的早熟苹果价格低于去年，采购商拿货谨慎	0
苹果是我国传统的水果出口品种之一，资源丰富，其产量占据全球半壁江山	0
我国的苹果出口国主要面向东南亚，比如泰国、孟加拉、印尼、越南等国家，今年受到疫情影响，1-2月基本无出口，3月恢复物流以来，出口数量和金额均处于历史同期低位，下半年希望有所改善	0
“倒春寒”影响有限苹果的实际减产小于预期	0
受到传统因素制约，我国苹果价格一直处于低迷状态，丰产时则大量降价，持续走低	0
我国苹果市场品种结构单一，质量层次不齐，中高端品种较少，低端品种大量占据市场	0
此外，苹果市场供给能力不稳定、产量波动较大、过度依赖于天气及产业化程度不高等因素都是影响苹果行业稳定发展的不利因素	0
种植苹果的主要成本来源包括种植成本和冷藏存储费用	0
2018年苹果价格大幅上涨，主要是受到“倒春寒”的影响，2018年的极端天气把苹果推高到11000元/吨，资金纷纷涌入	0
苹果价格上涨点燃了果农种植的激情，又再度出现过度种植，果贱伤农的情况	0
从数据来看，2019年度，苹果的种植面积在196万公顷附近，苹果产量在4100万吨左右，产量较2018年上升了24%	0
2020年，虽然之前炒作的天气带来减产，但其实际影响比预期中小很多：2020年4月，在北方发生的“倒春寒”主要影响地区为陕西、甘肃，市场预期冷冻天气将影响果树后期坐果，但实际对苹果减产影响小于2018年，苹果库存充足	0
苹果交易交割新规利空远月合约	0
2020年7月14日郑商所发布通知，重点对苹果期货交割质量标准、合约交割月份、仓库交割和车板交割的要求及流程、交割单位和交割延误滞纳金等方面进行了修订，由于交易交割细则变化较大，并涉及到很多方面，但总体看来，新规中对苹果期货交割品质量容许度要求放宽；降低可溶性固形物要求，包括允许冰糖心等品种交割，其实是扩大了苹果的交易交割范围，从之前的优质果变得更加平易近人，更接近商超的中档货源，此修改自苹果2110合约开始实施，也就是说苹果后期的交割品范围扩大，利空期价	0
综上所述，目前苹果处于供过于求的大背景下，现货价格偏空，投资者谨慎操作，不建议其盲目抄底	0
从业以来专注于白糖、苹果的分析研究，对白糖、苹果产业链分析有独到见解	0
