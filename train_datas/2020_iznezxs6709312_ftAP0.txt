郑商所“保险+期货”项目累计承保苹果现货12.7万吨	0
<pclass='ori_titlesource'>原标题：郑商所“保险+期货”项目累计承保苹果现货12.7万吨来源：新华财经	0
新华财经郑州12月13日电（记者李丽静）记者12月13日从郑州商品交易所（以下简称郑商所）获悉，2018年苹果期货上市以来，郑商所将甘肃纳入苹果“保险+期货”试点范围，截至目前共开展试点项目12个，累计承保苹果现货12.7万吨，覆盖8.5万亩苹果地，总保障金额近10亿元	0
甘肃省地处黄土高原、青藏高原和内蒙古高原交汇地带，山脉纵横交错，海拔相差悬殊，正是这种山地型高原地貌的气候环境使得甘肃成为中国“最甜”苹果的产区之一，甘肃多地都是国家苹果的优势产区，苹果产业成为当地的支柱产业，如何避免价格波动对产业带来的风险成为甘肃苹果产业面临的重要课题之一	0
在中国证监会指导下，郑商所于2016年开始苹果期货的研究	0
尽管面临苹果产业标准化程度相对较低、国际期货市场上无鲜果期货经验可借鉴等困难，郑商所按照既方便市场主体参与又严格保障市场安全运行的原则，完成了苹果期货的研发设计，并于2017年12月22日挂牌上市，苹果期货成为全球首个鲜果类期货品种	0
苹果期货上市后，郑商所在甘肃进行了12个苹果“保险+期货”试点项目，覆盖当地8个苹果主产县，惠及农户超过2万户，其中1.4万户（占比73%）为贫困户	0
参与项目的静宁县、秦安县、麦积区等均为国家级贫困县，自项目实施以来为当地苹果种植户提供了坚实的价格风险保障，助力脱贫攻坚效果显著	0
秦安县副县长李虎介绍，2018年11月初，太平洋保险公司联合东证、宏源、申银万国、渤海、华龙、华泰6家期货公司共同向秦安县推广苹果“期货+保险”项目，由政府配套资金、郑商所补贴、6家期货公司自筹支付保费，为秦安县4440个建档立卡贫困户的8880亩苹果提供保障，总保额超过1亿元	0
2019年初，苹果市场价格大幅波动，太平洋保险公司甘肃省分公司与各期货公司积极对接，启动理赔程序	0
“苹果‘保险+期货’项目的成功试点，让参保受益的贫困农户大受鼓舞	0
”李虎介绍，“保险+期货”项目的开展进一步提高了苹果产业抵御市场风险的能力，完善了产业发展的保障服务体系，探索出了一条金融扶持产业发展、助推脱贫攻坚的新路子，不仅得到了广大农户的充分肯定，同时也得到了甘肃省委领导的高度赞扬	0
郑商所总经理鲁东升表示，甘肃苹果“保险+期货”项目效果明显，果农收入得到保障，苹果产业稳定发展，在甘肃脱贫攻坚方面发挥了重要作用，也必将在助力甘肃乡村振兴方面发挥积极作用	0
