探访丰县苹果矮化栽培标准化试点项目基地	0
<pclass='ori_titlesource'>原标题：探访丰县苹果矮化栽培标准化试点项目基地	0
<divclass="img_wrapper"><imgstyle="max-width:640px"id="0"src="http://n.sinaimg.cn/spider2020529/789/w500h289/20200529/e480-iufmpmn8915549.jpg"img-size="500,289"/><spanclass="img_descr">正在给苹果套袋的女工	0
5月27日，丰县现代农业（果树）科技综合示范基地里，150亩“八月富士王”早已果实满枝，风儿吹过，满目都是青青的小苹果	0
这里也是省级农业标准化示范项目“苹果矮化栽培标准化试点”的研究示范园	0
果园里，穿着长褂、带着太阳帽、做足了防晒的女工们正挨个儿给青苹果套袋	0
项目承担单位——江苏大沙河集团的农艺师张忠伟正在观察苹果的长势，并不时和女工们交流着套袋的技巧：如果套袋时发现苹果长得太密，一定要先疏果再套袋，每个苹果间至少要留出10厘米以上的生长空间	0
园子里的苹果树都不高，矮点的枝条，女工们站在地上即可完成套袋；树冠部分，踩上1米多高的小型人字梯，一个个青苹果都能尽归手底	0
“矮化型苹果树一般在3.5米高，而正常生长的乔化苹果树可达10到15米高	0
”张忠伟说，苹果树虽然矮了，但产量并没有矮，反而更高了	0
原来，矮化型苹果树虽然躯干矮了，但高纺锤型树形的培育，让可挂果的枝桠更多；加上缩短株距、扩大行距（4米）的密植栽培法，让每亩地可栽植的苹果树达到100棵左右，是乔化苹果树的两倍	0
据了解，苹果矮化栽培是国际上较为先进的一种栽培技术	0
为此，2016年，丰县“苹果矮化栽培标准化试点”获省立项，今年4月，该项目通过了省市场监管局组织的省级验收	0
3年多的项目建设期里，在徐州市市场监管局、丰县市场监管局、丰县农委的协助下，徐州市果树研究所、徐州市果树站、丰县林业技术指导站、江苏大沙河集团等单位的8位果树农艺师（推广研究员）合力探索矮化苹果栽培标准化技术体系建设	0
通过这2150亩示范园的种植，项目组建立了丰县自己的矮化苹果栽培技术规程，形成了矮化苹果栽培技术的相关企业标准，并正申报省级地方标准；形成了宽行密株高支架立柱式栽培、水肥一体化、机械化打药等优质安全生产技术6项；带动全县推广苹果矮化栽培标准化生产基地近3万亩；示范区亩产7000斤果品以上，产品质量安全合格率100%，优质果率85%以上，亩效益1万元以上，实现企业年增加效益2000余万元以上	0
“丰县是果品大县，全县的果品种植面积超过40万亩，苹果种植达到了30余万亩，苹果产量占到了全省的70%左右	0
”丰县市场监管局副局长刘宇红说，矮化苹果标准化示范项目的实施，填补了省内无矮化苹果生产标准的空白,促进了矮化苹果新型栽培技术的发展，有力推动了丰县果业的提档升级	0
