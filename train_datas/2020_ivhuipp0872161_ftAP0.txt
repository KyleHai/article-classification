郑商所：进一步发挥苹果期货惠民安农作用	0
<pclass='ori_titlesource'>原标题：郑商所：进一步发挥苹果期货惠民安农作用来源：原创	0
会上，甘肃静宁苹果“保险+期货”县域全覆盖项目正式签约启动	0
郑商所相关负责人表示，本次论坛旨在发挥“稳企安农护航实体”品牌效应，发挥苹果期货在惠民安农方面的积极作用，探索交易所、期货公司与当地政府联动机制，建立期货农业经济发展新模式	0
平凉市常务副市长叶剑芳在会上介绍说，平凉市静宁县是农业农村部划定的全国苹果优势产区之一，果品产业经过连续30多年的发展，果园总面积稳定在百万亩以上，是全国苹果规模栽植第一县，被中国果品流通协会评为“中国苹果之乡”	0
“静宁苹果”被国家市场监督管理总局评为“中国驰名商标”，品牌价值达到134亿元，2019年全县果品产量88万吨，实现产值39.6亿元，人均果品收入5800元，苹果产业已真正成为助农增收致富的“聚宝盆”	0
特别是今年一季度疫情的影响下，当地苹果产业遭到冲击，果农收入受到不同程度的影响	0
据介绍，由郑商所支持，静宁县政府指导，人保财险、银河期货、大地期货、南华期货共同承办的苹果“保险+期货”试点项目覆盖了静宁县域全部的苹果种植贫困户，以及3个深度贫困村和部分龙头合作社，参保果农共计5903户	0
“今年的项目投入保费2256万元，为静宁县5900多户苹果种植未脱贫户、边缘户、监测户及相关果农承保苹果4.8万吨	0
该项目的实施，必将进一步打通静宁苹果产业与资本市场的对接渠道，推进苹果产业深度开发，带动果农保收增收、脱贫致富	0
郑商所相关负责人在会上表示，我国苹果主产区与重点贫困区域高度重合，开展苹果期货交易是期货市场落实国家脱贫攻坚战略的重要举措	0
苹果期货自2017年年底上市以来，运行质量稳步提升，改变了苹果市场长期缺乏权威、透明市场价格的局面，也为生产经营主体管理价格风险提供了工具，为利用“保险+期货”模式直接服务果农稳产增收创造了条件	0
今年2月静宁苹果在新三板上市，成为陇东地区首家上市公司，这标志着品牌价值139亿元的‘静宁苹果’正式踏入中国资本市场	0
据期货日报记者了解，郑商所2016年至2019年累计投入1.3亿元，支持开展了75个“保险+期货”试点项目，承保棉花4.1万吨、白糖33.3万吨、苹果5.7万吨，覆盖新疆、甘肃、广西、云南等11个省（自治区），50余个县（团场），其中32个为国家级贫困县，惠及农户10万余户，累计赔付金额超1亿元	0
