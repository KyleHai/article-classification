苹果：减产预期有所收敛，远月高点或已出现	0
<pclass='ori_titlesource'>原标题：苹果：减产预期有所收敛，远月高点或已出现来源：文华财经	0
征稿(作者：方正中期期货侯芝芳王骏）--供应端高库存已是市场共识，不过由于山东库存中果农货占比仍然较高，后期存在果农货集中出售的压力，旧果需求端则面临疫情好转带来的增量与时令水果增加带来的减量博弈，目前尚看不到较大的转机，旧果仍然是供应宽松格局，价格整体仍是偏弱表现，不过果农的销售对价格运行节奏有着重要影响，苹果07合约预期震荡偏弱走势	0
新季苹果进入幼果期，价格影响因素仍然是以供应端为焦点，天气对于市场的扰动仍存，不过目前尚看不到形成大面积扰动的基础，同时考虑到果树自身存在一定的修复能力，4月冻害对于新果影响预期可能会收敛，对于10合约来说，上周9409元/吨或已形成年度相对高点，除非未来出现重大影响产量的因子，否则价格继续向上的空间有限	0
全国苹果库存延续高位果农出货意愿为市场重要关注点	0
全国苹果库存消耗存在由西向东的特点，随着西部地区苹果库存的不断下降，苹果供应的关注点逐步向山东转移	0
卓创咨讯数据显示，截至4月30日，全国苹果库存为516.8万吨，大幅高于去年同期的199万吨以及2018年同期的321万吨，其中山东地区苹果库存为243.6万吨，占比为47.14%，陕西地区苹果库存为111.75万吨，占比为21.62%	0
2月由于新冠疫情的影响，苹果购销处于停滞状态，全国苹果库存消耗量降为近五年最低水平，3月随着国内疫情的好转，全国有序复工复产，产销区苹果补库需求加快全国苹果库存的消耗速度，全国苹果库存消耗量上升至近五年最高水平，进入4月，产销区补库需求完成，全国消费由于海外疫情形势的影响仍处于缓慢回升态势，苹果走货速度有所下降	0
卓创资讯数据显示，2月全国苹果库存消耗量为15.07万吨，大幅低于近五年均值80万吨，3月全国苹果库存消耗量为206万吨，大幅高于近五年均值106万吨，4月全国苹果库存消耗量为158万吨，略高于近五年均值145万吨	0
分地区来看，山东地区4月苹果库存消耗量为77.7万吨，高于去年同期的36万吨，也高于2018年同期的71万吨；陕西地区4月苹果库存消耗量为47万吨，高于去年同期的25万吨，低于2018年同期的55万吨	0
海外多国出现疫情拐点，苹果出口消费或逐步恢复	0
海外新冠疫情形势严峻，部分国家为了抑制新冠疫情的蔓延，采取封国的措施，疫情大背景下，我国苹果出口消费也受到了一定的影响，一季度苹果累计出口量处于相对低位，海关总署数据显示，我国苹果一季度累计出口量为24万吨，低于近五年均值30万吨	0
随着各国防控措施的加大，海外多国出现疫情拐点，后期随着海外疫情不断好转，我国苹果出口消费也将有所恢复，部分提振苹果需求	0
时令水果逐步增加苹果消费竞争加大	0
农业农村部的数据显示，4月末全国7种重点监测水果平均批发价为5.93元/公斤，相比月初的5.8元/公斤上涨0.13元/公斤，水果大类红富士苹果和柑橘类价格小幅上扬，4月末富士苹果批发平均价为6.95元/公斤，相比3月底的6.83元/公斤上涨0.12元/公斤，4月末蜜桔批发平均价为5.3元/公斤，相比3月底的4.74元/公斤上涨0.56元/公斤，除此之外时令水果菠萝价格也出现小幅上扬，由3月底的3.35元/公斤上涨至3.5元/公斤	0
进入5月份，水果的种类以及数量都呈现增加态势，除了原本的时令水果芒果、草莓、菠萝之外，芒果、桃子、火龙果等时令水果也陆续上市，整体水果的供应量相比4月进一步增加，储存类水果苹果优势进一步弱化，其他水果对苹果的替代消费或进一步增加	0
新季苹果仍处于生长期前期冻害对市场的影响仍在	0
5月份新季苹果进入幼果期，产区气温以及降雨量仍是市场的主要关注点	0
最新的天气预报来看，5月全国气温将逐步回暖，产区出现零下气温的概率降低；5月山东地区将迎来降雨，目前产区整体墒情正常，暂时看5月出现异常气候的概率降低，气候对于苹果生长的干扰也有所降低	0
4月产区气温波幅较大，部分地区再现零下气温，对于正值花期或幼果期的苹果以及梨产生了较大的影响，其中陕甘部分地区的苹果以及河北部分地区的梨冻害影响明显，市场减产忧虑升温，初步调研结果来看，此次冻害对梨的影响较大，对苹果的影响相对有限	0
睿惠咨询的信息显示，河北地区梨减产预估50-70%；卓创资讯的初步调研数据来看，陕西地区减产预估18.55%，甘肃地区减产预估为34.6%，两省苹果受损情况对全国产生影响大概在8%左右	0
作者简介：侯芝芳，应用统计硕士，现任方正中期农产品研究员，主要研究品种：菜粕、苹果	0
熟悉油脂油料以及苹果产业链，对菜类以及苹果有自己的分析框架，曾多次实地走访油脂油料相关企业，擅长数理统计分析以及基本面数据挖掘	0
