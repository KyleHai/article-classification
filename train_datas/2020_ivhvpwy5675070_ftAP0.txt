方正中期：双节备货提振红枣苹果市场短期价格延续震荡偏强	0
今年8月下旬至9月初以来，国内红枣和苹果价格持续下跌，红枣期货主连合约价格跌至8930元/吨附近、苹果期货主连合约价格也是大幅下挫至6662元/吨，主力合约换月后的期货价格均出现大幅反弹	0
我国传统双节——中秋节和国庆节临近，节前红枣苹果等水果备货及主产区恢复供应大大提振市场人气和市场价格，“金九银十”的传统旺季将有助于红枣和苹果价格延续震荡反弹走势	0

8月份以来苹果10合约跌势明显放缓，偶有反弹走势，但是迫于基本面的弱势背景，反弹力度都较弱	0
近期伴随着冷库苹果价格的回升，台风多次侵扰主产区以及双节备货启动等利多也有所浮现，9月8日苹果10合约暴力涨停	0
冷库苹果成交好转，成交价格回升	0
卓创资讯的信息显示，由于冷库苹果口感好于早熟富士，客商采购积极性较高	0
近期“三台”陆续登陆东北，对苹果部分产区产生一定的干扰，山东烟台地区由于台风的影响出现了6-7级大风，同时还伴有暴雨，对正值生长尾期的苹果产生了一定的影响，不过考虑到台风每年都有，且目前并未看到相关的特别大的灾害消息，暂时还是以合理区间波动来看，对全国苹果产量的影响预期较为有限	0
销区走货有所好转，由于时令水果进入秋季后相比夏季有所减少，销区对于苹果需求有一定幅度的好转，再加上双节临近，备货需求进一步提振销区的走货	0
不过目前走货好转主要还是基于环比的角度来看，同比来看仍然较为一般，后期随着双节备货的结束以及苹果供应的增加，走货速度或会恢复之前的缓增态势	0
综合来看，双节备货对红枣以及苹果的提振以阶段性为主，备货期间或会通过现货价格上升传导推动期价上涨，不过随着备货的结束，红枣以及苹果都将迎来新果的集中上市，再加上目前整体需求弱势的偏空仍未出现大的好转，因此红枣以及苹果整体预期都将呈现先扬后抑的宽幅区间波动	0
