苹果：需求羸弱，短期期价易跌难涨	0
<pclass='ori_titlesource'>原标题：苹果：需求羸弱，短期期价易跌难涨来源：文华财经	0
征稿(作者：徽商期货洪敏）--每年必备的天气炒作环节暂时告一段落，市场逻辑理性回归至苹果自身供需基本面，盘面也可以看出，价格不断承压下行，但是并不是因为今年苹果没有减产，而是减产的幅度远远不能抵消需求的下滑程度，供过于求仍然是短期走势的主旋律	0
但是仍然有风险点需要注意，今年的苹果质量可能不容乐观，交割炒作情绪或仍在埋伏，以及疫情好转后需求恢复，都是潜在的利多因素	0
产地行情：山东地区苹果主流成交价格走低，客商货源价格混乱，以质论价，果农货当前主流交易价格80#以上2.0元/斤左右，高价已经很少	0
销区行情：批发市场整体出货速度不快，近段时间因天气比较热，市场对瓜类需求尚可，苹果等水果需求一般	0
再加上时令水果价格大幅低于苹果，市场苹果销售略显困难	0
时令水果价格低，冲击苹果销量，客户采购谨慎	0

目前全国冷库存储苹果去库存率为68.6%，山东冷库库容比为36.1%，苹果行情较差，主流价格下滑，卖货存储商也有所增加，但是，采购客商认为后期行情还有继续下滑可能，近期采购积极性比较差，成交速度偏慢	0
本周正值端午备货阶段，本应对一潭死水的苹果行情有所提振，但是北京地区疫情苗头再次引发市场担忧，冷库苹果最后一个销售节日也难有表现	0
图2：苹果主产区库容比	0
据卓创数据统计，2020年全国苹果产量初步预估减产幅度为10%-15%，当然样本统计上可能存在差异，最终数据仍要等套袋完全结束后进行修正	0
但是受到公共卫生事件的影响，市场购买能力明显下降，市场采购人员同比往年减少较多，新季苹果需求仍然不被看好，加之近期时令水果价格比往年大度降低，晚熟苹果开秤价格预期较低	0
图3：全国苹果减产情况汇总	0
时令水果上市，尤其是在行情不佳情况下，苹果的价格相比其他水果比较高，对苹果冲击比较明显	0
今年水果总量供应充足，下游消费能力偏差，价格比往年同期低，晚熟苹果的开秤价格同比或要大幅降低	0
近期早熟苹果逐渐供应市场，当前上市品种主要是藤木一号	0
疫情以来，冷库苹果的销售节奏完全被打乱，本就增产的2019年度，需求还得不到充分释放，高库存难以避免，清库压力下后期价格仍然有继续下滑可能	0
新季度苹果的天气炒作也暂时告一段落，减产幅度逐渐浮出水面，有减产但力度不够	0
