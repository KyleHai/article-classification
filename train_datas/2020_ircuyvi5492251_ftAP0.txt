安塞区开启苹果产业智慧质量溯源体系建设	0
<pclass='ori_titlesource'>原标题：安塞区开启苹果产业智慧质量溯源体系建设	0
5月25日，安塞区农业农村局与平安保险安塞公司、陕果集团安塞分公司负责人共同签订了平安保险助推安塞区苹果产业智慧质量溯源体系建设项目合作协议	0
项目建成后，将实现苹果全产业链的信息追溯、监测	0
此次项目签约是苹果产业由政府推动变为市场撬动的创新之举	0
据了解，该项目通过“保险+科技”手段，聚焦解决农业产业链前期、中期和后期的核心发展难题，利用区块链、大数据等创新科技搭建智慧质量溯源系统，针对不同苹果产品实现全产业链溯源记录，帮助生产企业和种植户实时了解苹果生长期的状态，并为每一亩苹果地建立线上生产日志，记录操作人员栽植、灌溉、施肥、修剪、采摘等生产加工过程	0
安塞区农业农村局局长周赟表示，通过“保险+科技”模式，不仅降低了自然灾害风险，而且对进一步推动安塞区苹果产业转型升级具有重要意义	0
目前，安塞区苹果种植40万亩，其中挂果面积21万亩，涉及果农21240户	0
