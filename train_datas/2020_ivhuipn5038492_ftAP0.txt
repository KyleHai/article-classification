苹果：冷库果和早熟果“恶性竞争”，悲观预期下盘面仍将承压	0
<pclass='ori_titlesource'>原标题：苹果：冷库果和早熟果“恶性竞争”，悲观预期下盘面仍将承压来源：文华财经	0
征稿(作者：徽商期货洪敏）--当前已经来到7月下旬，时令水果逐渐减少，市场苹果需求本该有所恢复，但是冷库余货仍旧较多，冷库苹果价格不断走低导致早熟果开秤价格明显低于往年同期，早熟苹果货源供应量仍在逐渐增加，价格因此也不断滑落	0
二者互相影响造成情绪践踏，晚熟开秤价格的悲观预期笼罩市场，三四月份天气炒作带来的苹果涨幅已经收复殆尽，暂无利多支撑的苹果现货市场，悲观的市场情绪仍未释放完毕	0
目前全国冷库存储苹果去库存率为81.65%	0
批发市场整体出货量有所提高，恢复经营苹果生意的商户也逐渐增加	0
图1：苹果主产区库存变动情况	0
冷库苹果和早熟果互相影响低迷行情有继续往后传导可能	0
早熟已经开始供应市场，尤其是陕西地区苹果品种较多，早熟品种供应量也比较充足	0
今年早熟苹果受市场大环境影响，开秤价格低于去年同期，并且多数品种开秤之后，随着供应不断增加，价格呈现下滑趋势	0
表中早熟苹果平均价格为1.78元/斤，同比2019年下滑0.83元/斤，同比低31.66%	0
特别是西北地区，苹果同比去年跌幅比较大，其主要是因为2019年受前期高价影响，早熟苹果以及晚熟富士收购价格均比较高	0
库存货源与新产早熟之间相互竞争，低价冷库货不利于早熟苹果销售	0
时令水果品种减少叠加气温较高一定程度上利于苹果销售	0
时令水果品种已经开始大幅减少，当前阶段主要是桃子和西瓜，桃子价格变动不大，对苹果影响较小	0
西瓜价格普遍较低，货量较大，对苹果市场需求影响比较明显	0
但是由于近期多地降雨，气温不高，在一定程度上利于苹果销售	0
2020年度苹果市场持续处于低迷状态，归根结底就是苹果自身供过于求的供需结构，疫情还打乱了苹果的销售节奏，导致库存迟迟得不到消化	0
需要知道的是，新季苹果和旧季苹果并不是完全独立的，冷库苹果低迷的行情直接影响到早熟苹果的开秤价格，早熟果自身又不耐储，此时又面临冷库苹果的销售竞争，库存累积的情况下价格继续滑落概率较大，而这又会进一步影响晚熟苹果的开秤价格，整个市场已经被悲观预期所笼罩，情绪释放完毕仍然需要时间	0
短期来看，盘面仍会偏弱运行，建议逢高沽空为主，也可以尝试苹果10-01反套	0
