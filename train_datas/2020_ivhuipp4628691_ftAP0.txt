水果车侧翻苹果撒满地高速交警及时援手挽损失	0
<pclass='ori_titlesource'>原标题：水果车侧翻苹果撒满地高速交警及时援手挽损失	0
9月14日22时20分许，京港澳高速649公里路段，大量苹果散落在车道内，一部分直接被过往车辆碾碎，原来是一辆拉水果的货车因为爆胎侧翻所致	0
民警在现场见到，散落的苹果大部分都在右侧车道和应急车道，绵延数十米，由于未在路上设置警示标志，有些大货车避让不及便直接碾着苹果通行，爆胎的小货车侧翻在应急车道内，所幸司机没有大碍，只是看着满地的苹果欲哭无泪	0
见此情况，民警迅速在事故后方设置警示牌，避免发生二次事故，同时协同道路施救人员，大家齐心合力将散落的苹果及时收拢搬运，尽力挽回司机的损失	0
清理现场、搬运苹果、车辆施救及交通疏导等工作紧张有序进行，20分钟以后，现场清理完毕，道路恢复畅通	0
经调查，这辆小货车是从河北邯郸拉苹果回信阳，在行驶至事发路段时左后轮因老化突然爆胎，导致车辆侧翻，苹果散落，好在人员无大碍	0
