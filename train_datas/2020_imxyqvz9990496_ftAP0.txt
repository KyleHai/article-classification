期货助苹果产业主体保收入稳经营	0
<pclass='ori_titlesource'>原标题：期货助苹果产业主体保收入稳经营来源：原创	0
新冠肺炎疫情叠加高库存，近期苹果价格堪忧	0
据期货日报记者了解，由于2019/2020产季苹果产量恢复正常水平，截至2020年1月31日，全国苹果库存高达900万吨，处于近年来同期库存最高位	0
这种情况下，期货市场规避风险的作用更加凸显，部分果企利用期货市场进行套期保值、基差点价等，挽回了苹果价格下跌带来的损失	0
与此同时，2019年甘肃省静宁县苹果“保险+期货”县域全覆盖项目正在陆续赔付，赔付金额1600余万元，这份价格保险既有效缓解了“果贱伤农”，也保障了当地苹果产业平稳发展	0
烟台联盛果蔬有限公司（简称联盛果蔬）是一家集农副产品收购加工、冷藏保鲜、出口销售为一体的农业产业化企业，红富士苹果销售占很大业务比例，同时，该企业也是郑商所指定的苹果期货车船板交割机构	0
据了解，联盛果蔬担心后期苹果价格下跌，春节前就利用期货市场套保，卖出苹果期货合约80手（800吨）	0
春节期间，联盛果蔬认识到在疫情下后期苹果可能出现价格下跌、销售不畅局面，为管理企业经营风险，增加销售渠道，该企业在春节后开市第一天就再次卖出苹果期货合约100手（1000吨），较好地挽回了因苹果价格下跌带来的损失	0
栖霞市沃尔德果蔬冷藏有限公司（简称沃尔德）兼做苹果内销和出口贸易	0
春节前，沃尔德同样感受到了高库存压力大，该企业负责人告诉记者，他们预判春节后苹果价格可能下跌，于2019年12月前后在期货盘面卖出套保200手（2000吨）	0
早在2018年，莱州天赐宝物产有限公司（简称天赐宝）就尝试通过车船板方式交割60吨苹果，虽然数量不多，但是苹果期货上市以来车船板交割的第一单	0
“今年受疫情影响，西北地区尚有大量苹果存货未出库，结合交通运输情况，我们判断销售不太乐观，考虑到资金占用问题，决定采用基差点价的形式，依据当时的期货盘面价格签订1380吨苹果购销协议	0
近期，由人保财险联合银河期货、南华期货、大地期货共同实施的甘肃省静宁县苹果“保险+期货”县域全覆盖项目正在陆续赔付	0
该项目覆盖静宁全县8713名建档立卡贫困果农及3家合作社，保障面积3.5万亩，总投保苹果4.6万吨	0
银河期货相关负责人告诉记者，为保障贫困户利益，达到更好的赔付效果，也考虑到苹果行情多变的特点，项目组决定紧跟当地苹果销售节奏，分批入场出单，将保险期限覆盖整个苹果上市和销售时间，保单最晚于2020年的2月29日到期	0
该负责人介绍，去年11月中下旬，橘子、橙子、香蕉等水果大规模进入市场，但节日效应没有像往年一样体现，晚熟的高端苹果现货开始出现销售不畅，苹果价格不再坚挺，期货市场率先走出了下跌行情，此时，项目第二批入场对冲	0
2020年春节前后，疫情严重影响了苹果销售，静宁当地苹果销售完全停滞，部分贫困果农贮存在冷库的优质晚熟苹果，无法正常进入市场，现货价格、期货价格齐跌	0
