<pclass='ori_titlesource'>原标题：王东英种植的寒带区苹果走进长春农博会	0
近日，清河区苹果种植大王王东英带着他种植的新品种寒带区苹果和盆栽苹果树亮相第十九届长春农博会，将清河区的寒带区苹果带出辽宁，送进外埠市场	0
记者来到王东英的展位前时，王东英正在向参会群众和参展客商介绍寒带区苹果的优良品质、种植优势、种植技术和病虫害防治等	0
低矮的盆栽苹果树也能结出个头儿硕大的苹果，而且色泽艳丽美观、口感味道甜脆	0
王东英的新品种寒带区苹果成熟早、口感好、糖度大等优势引得人们纷纷前来品尝、求购，并有部分客商与王东英达成了合作意向	0
王东英说，“此次参展的目的就是要将自家的苹果推向外埠市场，在获得更大效益的同时，能够与其他参展商共同学习研究先进的苹果种植技术，为自己不断发展壮大奠定基础”	0
王东英表示：将以这次农博会为契机，加快转变发展方式，深化交流合作，加强苹果品牌建设，切实提高寒带区苹果种植的现代化水平，为清河进一步发展壮大做点贡献	0
