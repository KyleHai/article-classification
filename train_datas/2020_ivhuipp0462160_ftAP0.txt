苹果期货交割标准引领产业科技创新	0
<pclass='ori_titlesource'>原标题：苹果期货交割标准引领产业科技创新来源：原创	0
8月21日，来自全国主要苹果产区的产业企业及苹果质检机构、苹果期货交割厂库、投资投构等的100多名代表，在烟台参加了有关苹果期货优化交割流程的会议，同时共同见证了全球第七代自动化苹果分选线在齐鲁泉源供应链有限公司（下称齐鲁泉源）的试运行	0
当科技创新遇上苹果期货，两者碰撞出了绚丽的火花	0
齐鲁泉源总经理姜晔告诉期货日报记者，我国苹果产业独特性较为突出，加上行业基础相对薄弱，要想让苹果产业长期保持稳定发展的良好势头，不断提高广大果农的收入水平，特别是打破传统的、固化的、相对落后的市场收购、品质确定与定价模式等，必须要在苹果期货市场的引领下，加快苹果产业链各环节上的科技创新、加快产融融合的步伐，逐步形成标准化、规范化、规模化等苹果产业发展模式	0
据记者了解，苹果在我国水果产业中具有重要地位	0
在农业农村部认定的122个苹果重点县（市）中，曾有33个国家级贫困县，涉及贫困人口1751万人，苹果是当地农民的重要收入来源，苹果产业对促进当地经济发展，农民脱贫致富具有重要意义	0
与此同时，苹果作为鲜果，产业特性独特	0
一是现货标准化程度低，如同为红富士苹果，不同产地个性化差异大，质量判定主要看外观、靠经验，而外观指标如着色、瑕疵、果形等都是主观指标，同一批货物可能因个体判断差异出现不同的认定结果	0
二是苹果现货产业化程度低	0
目前，苹果分选基本靠人工，受工人分拣水平、判断差异的影响，货物质量参差不齐、一致性差	0
苹果产业企业基本都是中小民营家族企业，贸易量排名前十的企业行业占有率不足1%，其参与期货市场意识、能力偏低	0
苹果期货特殊的产业情况，给合约规则设计和市场运行带来较大的挑战	0
对每颗苹果的称重误差小于2—4克，能够严格区分出苹果的果径大小，不会出现人工分选因为视觉误差导致的果径窜级的问题	0
每颗苹果在通过照相机系统时，设备会让苹果旋转360度，同时拍摄120张照片，对果子的颜色，果面上的果绣，斑点，刺划碰压伤，揉伤能够做出精准的分辨和识别，然后电脑系统会把识别后的果子传送到不同质量标准的包装通道	0
原料上机部分，是通过水槽循环水带动，进入喷淋系统，让苹果的表面更加洁净，避免了果面粘了尘土树叶等杂质造成照相机系统的误判	0
姜晔认为，全球第七代自动化苹果分选线的试运行将为我国苹果产业未来发展产生重要影响	0
一是将打破传统的苹果收购模式，去掉中间人工分拣	0
以收购一二级“80苹果”为例	0
在传统这个收购链条里，代办点会想办法会压低果农的苹果价格，同时代办点人工分拣标准尺度弹性大，会产生碰压伤果和“人情果”，而冷库二次分拣会产生碰压伤	0
新设备投入后，果园粗选摘下来的苹果可以直接上机按设定的质量标准分选出来一二级“80苹果”，可以节省两次人工分拣费用，避免两次人工分拣产生的碰压伤	0
二是节省的费用可以补贴果农，可以更好地发挥苹果期货扶贫、惠农的作用	0
产业企业可以把节省下来两次人工费用全部返还给果农，每斤返还0.2—0.3元，意味着好苹果卖了好价格，增加了果农的收入	0
另外，降低了苹果下树收购环节的用工数量，还可以减轻体力劳动强度	0
最新一代的分选设备，从原料上机分选，到各个标准的识别，一直到进入识别后的各个通道，标准一致，全程直观可视，既不能以次充好，次果卖高价，也不能把好果子混到次果里，让好果卖低价，会让苹果交易的价格和交易的标准更加透明、果农卖的安心，采购商买的放心	0
例如：齐鲁泉源出口苹果的数量逐年递增，用工人数尤其到农忙季节十分紧张，自动化分选设备投入后，前端苹果的质量分选由机器完成了，末端只需装箱一个环节，大大节约人工，节省单箱装货时间，可以大幅提高公司的日出货量	0
作为生鲜的苹果，果品质量和标准在不同的采购渠道和销售通道里弹性比较大，买卖双方往往各说各的理，归根到底是没有一种让大家普遍认可的标准测定方式，传统经验和人为感官占据主导	0
