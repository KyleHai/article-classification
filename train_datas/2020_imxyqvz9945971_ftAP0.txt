疫情困住“西北货”，苹果“保险+期货”赔付果农1600万	0
<pclass='ori_titlesource'>原标题：疫情困住“西北货”，苹果“保险+期货”赔付果农1600万	0
2019年甘肃省静宁县苹果“保险+期货”县域全覆盖项目正在陆续赔付，赔付金额1600余万元	0
疫情叠加高库存，近期市场上的苹果价格堪忧	0
据了解，由于2019/2020产季苹果库存处于近年来同期最高位，截至2020年1月31日，全国苹果库存高达900万吨	0
这种情况下，部分果企利用期货市场进行套期保值、基差点价等，挽回苹果价格下跌带来的损失	0
今年受疫情影响，西北地区尚有大量存货苹果未出库，结合交通运输情况，销售行情不乐观，考虑到资金占用问题，公司决定采用基差点价的形式，依据当时的期货盘面价格签订1380吨苹果购销协议	0
山东地区苹果期货车船板交割机构—烟台联盛果蔬有限公司（下称“联盛果蔬”），是一个农副产品收购加工、出口销售的农业产业化企业，其中红富士苹果销售占很大业务比例	0
联盛果蔬也担心后期苹果价格下跌，春节前利用期货市场套保，卖出苹果合约80手（800吨）	0
春节期间，预计在疫情环境下，苹果可能出现价格下跌、销售不畅局面，在春节后开市第一天该企业再次卖出苹果合约100手（1000吨），较好的挽回因苹果价格下跌带来的损失	0
郑商所相关负责人告诉第一财经记者，近期2019年甘肃省静宁县苹果“保险+期货”县域全覆盖项目正在陆续赔付，赔付金额1600余万元	0
具体看来，该“保险+期货”项目由人保财险联合银河期货、南华期货、大地期货共同实施，覆盖静宁全县8713名建档立卡贫困果农及3家合作社，保障面积3.5万亩，总投保苹果4.6万吨	0
银河期货相关负责人告诉第一财经记者，2020年春节前后，疫情严重影响了苹果销售，静宁当地苹果销售完全停滞，部分贫困果农贮存在冷库的优质晚熟苹果，无法正常进入市场，现货价格、期货价格齐下跌	0
考虑到苹果行情多变的特点，项目组决定紧跟当地苹果销售节奏，分批入场出单，将保险期间覆盖整个苹果上市和销售时间，保单最晚于2020年的2月29日到期	0
