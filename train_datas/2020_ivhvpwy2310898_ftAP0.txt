<pclass='ori_titlesource'>原标题：对苹果产业发展及金融支持情况的调查——以甘肃省庆阳市为例	0
近年来，甘肃省合水县大力实施精准扶贫和乡村振兴战略，把苹果产业作为帮助农民增收致富的主导产业来抓，苹果产业已成为农民群众增收致富的重要支柱产业	0
合水县苹果产业发展现状及金融支持情况	0
合水县地处陇东子午岭腹地，自然资源得天独厚，生态良好，年总日照时数达到2376-2492小时，全县平均海拔1387米；年均气温9.6℃，年均降水量560.3mm，无霜期155天，境内光照充足，降水量充沛，昼夜温差大，耕地以黑垆土、黄绵土为主，土层深厚，土壤质地沙粘适中，是符合苹果生产7项气候指标的最佳优生区	0
全县苹果树以每年新栽1万亩速度递增	0
截至2019年底，全县苹果园留存面积达到9.01万亩，挂果面积6.14万亩，苹果产业覆盖了11个乡镇69个村，惠及建档立卡贫困户5414户22809人，贫困户果园面积26230亩	0
全县苹果产量从2013年的5.7万吨增长至2019年的9.45万吨，增长65.8%；产值由2.6亿元增长至3.78亿元，增长45.4%；提供农民人均可支配收入由1733元增长至2440元，增长40.8%	0
苹果已成为合水县农业的“当家树”、农村的“致富树”和农民的“摇钱树”	0
同时，加快有机苹果基地建设，主攻郁闭园改造，大力推广矮化密植建园	0
2002年，合水县“董志塬”牌苹果商标注册成功；2003年通过了省级无公害食品认证；2005年通过了国家绿色食品认证	0
2019年取得了国家市场监管总局“合水苹果”地理标志证明商标	0
与京东、阿里巴巴等国内大型电商企业合作，开展现场“秒杀”、淘宝双十一直播活动、阿里巴巴平台线上促销等活动，提升了合水苹果知名度和市场竞争力	0
近年来，合水县农村信用社、金城村镇银行等涉农金融机构围绕苹果产业发展链条，创新开办了果业通、金果贷、苹果仓单质押贷款等特色产业信贷产品，其中“苹果仓单质押贷款”在2017年荣获人行兰州中支“创新信贷产品奖”	0
截至2019年末，全县金融机构向苹果收储企业提供贷款存量1.31亿元，向苹果收购商贩提供贷款存量0.3亿元，存量农户小额贷款7.18亿元，有近四分之一用于农户果树栽培管护	0
合水县人保财险等4家保险机构开办了针对苹果生产保险业务，在受灾最严重的2018年理赔支出达393万元，保障了合水县苹果产业持续发展	0
合水县苹果产业发展遇到的主要问题和金融支持障碍	0
在生产布局上，塬面发展较快，山区果树发展速度相对迟缓;在品种结构上，一是主栽品种比较单一，新品种的更新换代慢；二是熟期搭配不合理，早、中、晚熟品种分别占3%、17%和80%，成熟期过于集中，苹果采摘后市场销售压力大；三是鲜食与加工品种比例不协调，主栽品种基本上都是鲜食品种，保鲜期短，无法实现长期保质和错峰销售	0
目前合水县苹果生产仍以小规模家庭果园为主，一些提高品质的综合措施没有大范围应用或应用不到位，产品质量很难达到高品质和标准化	0
大多数果农仍沿用传统的栽培技术和管理方法，在果园生产管理中没有完全掌握良种壮苗整形修剪、疏花疏果、平衡施肥、果园覆盖、节水灌溉、综合防治、果实套袋、摘叶转果等改善苹果品质的标准化栽培技术	0
合水县苹果一般的采后处理主要是贮藏和销售，缺乏先进地区生产的鲜食苹果经过机械清洗、打蜡、分级、包装后再投放市场的流程	0
（六）农业灾害频发，严重影响苹果产业的健康持续发展	0
苹果生产受气候等外界因素影响巨大，在现有条件下种植户还无法对自然灾害进行有效防范，成为影响苹果产业持续健康发展的最大障碍	0
如2018年，受低温冻灾天气影响，合水县苹果大量减产，许多果农颗粒无收，果库无果可存，许多收储企业和果农无法清偿银行贷款利息，造成大量贷款逾期，也限制了银行信贷持续支持	0
以创建“优质无公害苹果基地”为目标，坚持规模化、多样化的建设布局，在种类和品种的发展上既相对稳定，保证规模竞争优势，又及时调整，跟上市场需求	0
（六）创新开发苹果保险产品，增强苹果产业抗灾害能力	0
保险机构要探索开办更多种类的苹果灾害保险，分散苹果生产过程中的风险，保证果农的持续经营能力	0
（七）建立不良信用信息容错纠错机制，实现苹果产业链完整持续发展	0
