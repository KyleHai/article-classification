苹果：水果市场供应充足，期价或弱势为主	0
<pclass='ori_titlesource'>原标题：苹果：水果市场供应充足，期价或弱势为主来源：文华财经	0
征稿(作者：瑞达期货王翠冰）--2020年5月中旬以来，苹果期货主力2010合约持续下跌，跌幅超10%，主要受苹果冷库居高库存及需求持续疲软等影响	0
夏季时令鲜果逐渐大量上市，且价格相对低廉，进一步削弱苹果消费端，加之果农多数让价出售为主，预计后市苹果市场难有较大改观	0
今年苹果居高库存，且走货仍相对较慢	0
新季苹果处于坐果阶段受到冻害，以往不同的是，受害面积相对广，将对新果优果率造成的影响颇多	0
若按照市场综合预估减少10%左右来测算，远不及2018年度减产幅度，新季苹果产量大致在3950-4000万吨	0
此外，从旧年度的库存来看，全国苹果冷库库容量大致在1440万吨附近，2019年度全国苹果库容比为80%，高于去年库容比50%的水平，其中山东冷库库容量在410万吨，陕西库容为450万吨，山西库容在180万吨；截止6月下旬，山东库容比为35.0%，陕西库容比为17.0%，预计山东苹果冷库仍剩余约140万吨，陕西冷库苹果大致剩余70万吨，其中山东地区苹果总体行情呈现偏弱态势，客商数量比较少，后市冷库整体出货量继续走慢可能，供应压力加剧；西北地区冷库出货速度相对平稳，整体变动并不是特别大	0
下半年来看，与往年同期比较，今年冷库苹果库存较大，尤其是山东冷库苹果	0
随着国内气温逐渐升高，库内苹果品质出现虎皮等问题逐渐显现，货商及果农或急于让价销售可能性加大	0
加之早熟新季苹果陆续上市，且该年度水果多数处于充足状态，水果价格相对低廉，苹果出库速度仍较慢	0
主产区富士苹果现货价格纷纷下调为主	0
从苹果产区角度而言，山东栖霞纸袋80#以上一二级果农货苹果价格为2.00元/斤，相较于上个月的2.6元/斤下跌了0.60元/斤，跌幅为23.07%；山东沂源纸袋74#以上苹果价格为1.4元/斤，较上个月的2.00元/斤下跌了0.6元/斤，跌幅为30.0%；陕西洛川纸袋80#以上苹果价格为3.70元/斤，相较上个月保持不变；甘肃静宁纸袋75#以上苹果价格为3.40元/斤，相较上个月环比持平；整体产区苹果现货价格下跌为主，除了陕西产区价格相对坚挺，由于该产区处于优质苹果占比较大	0
三季度时令鲜果供应进入集中期，加之今年水果多数丰产，且价格相对低廉，受替代品价格下调，预计旧季苹果价格稳中下调为主	0

根据农业部监测的主要水果产品价格显示，截至6月下旬富士苹果批发平均价为3.56元/斤，较上个月的3.515元/斤，上涨了0.045元/斤；香蕉批发平均价为2.34元/斤，较上个月的2.72元/斤，下跌了0.38元/斤；鸭梨批发平均价为1.53元/斤，较上个月的1.58元/斤，下跌了0.05元/斤，上半年来看，香蕉、鸭梨批发价格均下调，下调幅度在0.05-0.38元/斤，整体上水果价格基本稳中下调为主	0
综上所述，今年冷库苹果库存延续高库存局面，尤其是山东冷库苹果货源充足且果农货占比较大，且随着早熟苹果上市，冷库苹果清库难度加大，供应压力加剧	0
整体上，今年疫情重创水果消费，且新上市水果价格相对低廉，苹果需求仍显疲软，预计后市苹果期价仍难言可观	0
