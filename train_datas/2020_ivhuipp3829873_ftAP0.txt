【农产品—苹果】双节将至+新果上市苹果行情仍然偏空	0
苹果期货自今年5月以来，走出一波持续下跌行情，前期一度刷新上市以来低点	0
但本周苹果期货主力合约本周累计上涨4.17%，周中最高报7358元/吨，走出一波小幅反弹	0
长江期货研究院黄尚海表示，近2月苹果行情分2个阶段：第一阶段，在7月低至8月中旬，夏季水果与早熟苹果大量上市，冷库苹果存量高于去年，加之果农对苹果价格后续走势较为悲观且急于清库，现货价格持续走低，苹果期货价格于8月7日达到低点6805元，10合约持仓达到14万手；随后受产西北产区天气影响，现货炒作嘎啦果质量受损，优果价格回升期货随之上涨至最高7300元/吨，空头资金避险，10合约持仓减仓至12万手	0
8月中旬，山东地区烟台等地嘎啦大量上市，价格较去年下跌1元/斤，现货整体价格不高，成交冷淡；8月下旬西部地区嘎啦交易接近尾声，有客商借机炒作，西部嘎啦价格略有上涨，后由于下游批发市场接受度不高，西部早熟富士开始上市，现货价格继续下跌，期货10合约在9月3日跌至6622元/吨，10合约持仓达到16万手，创10合约近两年新低；4-5日受台风影响，部分产区出现冰雹天气，加之夏季水果已经接近尾声，西部早熟富士交易接近尾声，山东地区红将军暂未上市，库存果数量不多，苹果市场出现青黄不接现象，现货价格随之上涨，西部早熟富士最高达到3.0-3.5元/斤	0
短期来看早熟苹果价格上涨对晚熟富士上市价格有支撑作用，待9月下旬西部晚熟富士开始摘袋，若无重大利多因素影响，苹果行情仍然偏空，建议投资者暂且观望或等行情明了进行可逢高沽空操作	0
