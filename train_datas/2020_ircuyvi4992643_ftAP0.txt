<pclass='ori_titlesource'>原标题：苹果期货首次厂库交割圆满完成来源：原创	0
本报讯（首席记者乔林生）5月22日，苹果期货上市以来的首次厂库交割在郑商所指定苹果交割厂库威海市翠虹果品股份有限公司（下称荣成翠虹）顺利装车发运，完成提货，苹果期货首次厂库交割工作圆满结束	0
“作为苹果期货上市以来的首次厂库交割，此次交割得到了当地政府、产业企业、投资机构以及会员单位的广泛关注	0
”荣成翠虹总经理岳建东告诉记者，荣成翠虹始终关注苹果期货的发展，认真研究苹果期货交割标准，将企业现行标准与期货标准相结合，促进企业现货业务、期货业务协同发展	0
此次厂库交割是对苹果期货交割方式的一次尝试，为荣成翠虹后续参与苹果期货交割业务提供了宝贵的经验	0
岳建东表示，荣成翠虹为了准确掌握苹果交割流程，在ap005合约上同时采用了厂库和车（船）板两种交割方式，均顺利完成交割	0
威海景星主要从事苹果进出口和商超渠道供货，其负责人吴海洲表示，此次交割货物的质量与下游商超的质量标准一致，交割货物将会供应至超市	0
两种交割方式适合不同类型的贸易方式，厂库交割适合厂库与终端商超直接对接，车（船）板交割则适合贸易商之间的相互调货，两种交割方式互为补充，为不同类型的产业企业参与苹果期货提供便利，可以更好的促进苹果期货的发展	0
