苹果新专利曝光	0
美国专利商标局日前公布了苹果一项名为“带有玻璃组件的手写笔”的新专利，涉及一款笔尖或当前平的侧面采用玻璃材质的applepencil	0
根据苹果的描述，目前采用的塑料和涂层材质通常是多孔的，很容易进入碎屑，而用玻璃材质更耐用，会防止碎屑的进入从而减少磨损	0
此外，苹果指出，玻璃的可塑性更强，可以根据个人喜好进行个性化定制	0
值得注意的是，玻璃材质的applepencil可利用触控笔内的电容式触摸传感器来促进对触摸输入的检测，能够更敏感地感受到用户的敲击手势、滑动手势以及滚动手势	0
据悉，苹果为applepencil申请了不少专利	0
例如此前集微网报道的根据内置系统改变applepencil握持区域的厚度或形状，以适应用户的绘画或书写习惯；设置多个触摸传感器用于检测使用者的手势，进而根据使用者输入的指令执行相关功能；此外还有专利显示applepencil可以向用户提供触觉反馈，同时还能够读取使用者抓握力度等握持方式数据	0
