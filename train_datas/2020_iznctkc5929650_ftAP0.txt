大连苹果地标品牌推选活动正式启动	0
<pclass='ori_titlesource'>原标题：大连苹果地标品牌推选活动正式启动	0
又是一年苹果红，10月正是大连苹果成熟上市之际	0
为加快推进大连苹果地理标志品牌建设，实现大连苹果产业高质量发展，由大连市农业农村局指导，大连市现代农业生产发展服务中心（大连市农业科学研究院）主办，大连美丽乡村文化研究院承办的大连苹果地标品牌推选活动即日起正式启动,将通过市民网络投票评选和专家评审的方式，最终将推选出“大连苹果十大好吃品种”“农产品地理标志大连苹果十大品牌”和20个“农产品地理标志大连苹果示范基地”	0
“大连苹果十大好吃品种”推选分组织筹备、申报、初评、展示投票、现场调研品鉴、审核确定等六个阶段	0
广大市民自今日起至10月17日，可登录大连发布和大连休闲游微信公众号，查询10月10日的文章进行投票，或者后台回复“苹果”，在活动页面选出你最喜爱的苹果品种，两个平台票数相加计算最后的公众投票分数	0
微信投票的同时，组委会组织相关专家和公众代表深入苹果主产区现场调研品鉴，围绕苹果品种的内在品质、外观色泽、文化提炼、经济效益等进行专业测评，最后召开专家评审会，按50%的公众投票分加50%的专家评审分计算参评品种总分，评选出“大连苹果十大好吃品种”	0
“农产品地理标志大连苹果十大品牌”推选的申报主体必须是大连苹果农产品地理标志使用企业，组委会将组织专家围绕品牌质量、生产情况、销售情况、产品认可度、品牌影响力等方面进行综合评价，按专家评审分计算参评单位分，评选出“农产品地理标志大连苹果十大品牌”；同时评选出20个“农产品地理标志大连苹果示范基地”	0
