原标题：天行健，“苹果”以自强不息	0
昨天是平安夜，看着一颗颗被装在精致包装里的高价“平安果”，再回首来看这一年里苹果行情的兴衰，先是在本该旺季销售的年初因封锁胎死腹中，之后再借霜冻之风假炒上涨，最后待新果真正上市之后却是无人敢接连续下探，苹果将这样在争议中跨过新年	0
今日苹果主力05合约报收6759，涨幅3.18%	0
从目前的销售情况来看，在其他替代水果的挤压下，苹果的销售更为缓慢且利润偏低，当下又正面临砂糖橘上市抢占的市场，且柑橘类水果产地的采购价格仍在持续下滑，销区也在不断走低的售价中打开市场，这也就是的很多终端客商不愿意去销售卖的慢，利润低的苹果，也让苹果变得越来越难卖，价格越来越难涨	0
截至上周五，富士苹果批发平均价格为3.58元/斤，较前周环比上涨0.08元/斤；蜜桔批发平均价格为2.06元/斤，较前周环比下跌0.209元/斤；香蕉批发平均价格为1.87元/斤，较前周环比下跌0.125元/斤	0
不过，需要注意的一点是，柑橘、柚子、砂糖橘等水果虽然在上市后会对苹果市场造成一定冲击，但还不会对苹果造成毁灭性打击，因为这些替代水果并不是只在今年今时才出现，而是每年都会如期上市，往年的苹果也都能正常销售	0
所以苹果销售情况不佳价格难涨或许另有原因	0
而消费模式的转变也就导致了苹果市场渠道出现变化，拥有对接渠道的批发客商会先积极售卖自己的存货，而为了减少自身的损失，在售卖自存货源时，即使销售缓慢，客商也不愿主动让价，这就使得虽然市场走货非常缓慢但市场价格却一直保持平稳	0
而另一边果农看着手里的苹果着急出货也愿意让价，但自身没有销售渠道，无法将降价的货源发至市场来影响市场价格，只能等有渠道的客商将自存货卖完后，客商通过主动压价来博取利润，而果农在高库存压力下，不得不低价抛货	0
西北减产山东果次，未来苹果仍有故事可说	0
截至12月16日，全国苹果冷库存储量为1169.35万吨，较上周减少9.07万吨，降幅为0.77%，同比去年降幅为1.38%	0
其中山东苹果冷库库容比为83.92%，同比增幅7.62%，陕西地区库容比为80.36%，同比减幅为6.97%	0
从库存可以看出，今年陕西及山东地区的苹果入库情况相较往年要更快也更多	0
一方面西北地区减产基本无疑，而对于往年正在热卖的地面果也是早早的收入库中以保证苹果质量，而在另一边的山东地区，虽然库存数量更高，但因其前期受冰雹袭击严重，优果率偏低苹果整体质量较差，所以在入库的货源中一二级商品货比例特别少下捡果比例要比普通年份高的多	0
另外，当下时间节点本就不属苹果销售高峰，而苹果又是入库便可卖一年的长线水果，虽然目前入库量巨大且走货缓慢，但对于苹果而言，未来还有一年的销售时间	0
综上所述，目前苹果市场乱局丛生，在各路水果一起上市且保持低价的挤压下，本就销售不佳的苹果再加上混乱的销售渠道，苹果价格一时难涨，但还不会置于绝境，今天苹果价格的反弹就能说明	0
另外，今年苹果减产已成事实，劣果次果比例明显增多，未来待苹果走货逐渐顺畅，质量差的苹果销售殆尽之时，市场就会发现枪里原来没有子弹了，好果价格自然会发生转变，未来苹果市场也会有故事可讲	0
