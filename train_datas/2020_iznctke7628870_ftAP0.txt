征稿(作者：徽商期货洪敏）--2020年季度苹果库存量再创近年来新高，当前本应是苹果消费旺季，但是冷冬下水果整体大环境低迷，苹果相较于其他水果价格偏高，受到的影响更为明显	0
产区走货缓慢，已有降价迹象，但仍无事于补，后市苹果的销售压力非常之大，继续降价是大概率事件	0

主要原因在于国内收购季自10月开始，价格明显高于9月份，高价难销，不利于出口，同时人民币升值明显，抑制苹果出口	0
柑橘价格继续下滑进一步影响苹果销售	0
秋冬季节苹果最大的竞争对手就是柑橘类水果，今年柑橘产量增加，价格本就低于去年同期	0
近段时间因市场走货速度缓慢，客商采购积极性降低，产地收购价格再次下滑，对苹果销量影响比较明显	0
后期新上市货量持续增加，货源持续时间可以至次年3月份，此期间对苹果均有较大影响	0
