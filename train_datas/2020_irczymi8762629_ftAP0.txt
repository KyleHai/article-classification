2020年4月28日，期货日报：郑商所开展苹果期货质检宣讲保交割顺畅	0
<pclass='ori_titlesource'>原标题：2020年4月28日，期货日报：郑商所开展苹果期货质检宣讲保交割顺畅来源：新闻信息部易盛	0
日前，郑商所组织指定苹果质检机构赴山东烟台等地开展交割宣讲活动，向相关车（船）板服务机构、产业企业宣讲苹果期货交割质量标准，保障苹果期货2005合约及后续相关合约交割工作顺利开展	0
据期货日报记者了解，郑商所对之前交割业务中买卖双方关注的重点指标进行了系统梳理，对市场关注的关键点，有针对性地开展此次苹果交割标准宣讲活动	0
指定苹果质检机构通过现场检验，对果锈、着色等检验尺度的指标着重讲解，通过对上述指标的现场模拟检验，各车板服务机构对苹果期货检验标准更加明确，也为即将到来的苹果期货2005合约交割做好了质检准备工作	0
郑商所要求质检机构加强内部管理和培训，准确地把握期货检验标准，规范苹果期货质检的各项流程，促进检验工作的有序开展	0
据了解，此前，为保障苹果期货交割业务在新冠肺炎疫情的特殊时期顺利开展，郑商所采取多项措施，提前布局、提前谋划，确保苹果期货交割顺畅	0
为确保苹果复检业务的及时开展，郑商所提前召开质检机构电话会议，总结质检工作中存在的问题，指出质检工作难点、要点和关键点，要求质检机构加强内部管理，优化检验流程，持续做好人员培训，切实有效提供高质量检验服务	0
郑商所要求相关会员单位，及时向参与苹果交割的客户讲解交割规则，提示车（船）板交割服务机构所在地疾控政策，协助客户办理交割手续；要求车（船）板交割服务机构及时与客户沟通并提供优质服务，讲解当地疾控政策，协助客户办理通行证，协助客户解决出行、餐饮、住宿等疫情防控期间所面临的实际困难	0
