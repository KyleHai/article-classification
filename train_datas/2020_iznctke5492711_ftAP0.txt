在供应宽松拖累下，苹果期价呈现不断下跌走势	0
展望后市，虽然在成本端支撑下，苹果2101合约继续下行的空间被压缩，但是如果需求未出现明显好转的话，苹果期价大幅回升动力也不足，大概率会维持低位区间运行	0
数据显示，洛川地区70#以上半商品开秤价为3.0—3.3元/斤，高于2019年同期，国庆节期间曾上涨至3.5—3.7元/斤；山东地区苹果上市时间晚于陕西地区，陕西收购价上涨带动山东地区果农惜售情绪，再加上优果率不佳等影响，收购价上市之初也呈现高开高走态势，山东栖霞地区80#以上一二级开秤价为2.8—3元/斤，月中旬的时候上涨至3.2—3.5元/斤	0
不过，产区收购价上涨并没有带动销区的走货情绪，随着山东产区苹果量的增加，收购热情下降，产区收购价格出现回落，尤其是12月23日霜降之后，收购价格的回落幅度将扩大，11月底山东产区80#以上一二级货源主流收果价格回落至2.6—2.8元/斤，陕西地区70#以上半商品收购价回落至2.9—3元/斤	0
今年新季苹果虽然出现了一定幅度的减产，但是入库量并未跟随下滑，反而创近五年新高，足见苹果市场供应宽松	0
数据显示，全国苹果今年入库量预估为1142万吨，其中陕西地区入库量接近323万吨，基本接近去年的水平，山东地区的入库量接近420万吨，为近五年的最高水平	0
今年高入库主要受以下三方面因素的影响：一是新果上市之初产销区供需情况不匹配，产区收购价高开高走，而销区价格并未跟随上涨，导致高价货源被动入库；二是上市之初收购价高企增加了果农的惜售心态，陕西果农入库意愿有所增强；三是客商分流以及炒作资金介入，由于西部优生区减产幅度较大，导致部分客商分流至山东地区收购，还有一些炒作资金看好行情介入苹果收购，山东地区客商货占比有所提高	0
苹果消费量同比下滑	0
数据显示，自10月中旬开始，市场整体走货速度明显放缓，加上柑橘低价上市，对苹果消费形成较大的冲击，多数批发市场整体销售量同比减少幅度较大，部分客商反映相比往年减少四至五成	0
综合来看，在水果疲弱的消费环境以及柑橘类对苹果消费冲击延续的背景下，苹果供应宽松格局短期内难以转变	0
不过，在苹果价格进入低位之后，成本支撑逐渐凸显，期价进一步下行空间将有限，在没有出现其他打破当前局面的因素的情况下，期价或呈现低位区间波动	0
