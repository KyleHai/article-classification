一路狂跌下的苹果还有回头路吗	0
自今年年初至今，苹果合约的价格如同坐了一趟过山车，先是受三四月份天气炒作带动，苹果价格直线飙升，五月份最高冲至9400点，但随着炒作情绪淡化，苹果库存堆积难清，苹果价格又开始一轮俯冲	0
到目前为止，在苹果主力将前期涨幅跌完以后似乎并没有收手的意思，主力10合约今日报收7352，下跌128，跌幅为1.71%	0
近期随着苹果价格的不断下调，山东地区走货速度明显增快，部分冷库为清库存选择低价处理	0
截至7月16日，全国苹果库存为224万吨，远远高于去年同期的88万吨以及2017年同期的157万吨，而五年内7-9月苹果消费量均值为135万吨	0
面对如此高的库存量，国内消费市场明显很难消化，为此苹果只能不断降价销售了	0

随着苹果价格的持续走低，客商采购积极性也逐步提高，而果农和出货商们多数也愿意顺价出货，不过当前库存剩余量仍然远高于往年同期，所以库存方面仍然面临很大压力	0
受库存苹果影响，早熟果销售受阻	0
就在库存苹果尚未出清的同时，早熟苹果也已经开始上市销售了，尤其是陕西地区由于苹果品种较多，早熟品种供应较为充足	0
但是，受今年苹果市场大环境影响，早熟果的开秤价格要低于往年同期，而且随着更多品种的开秤和供应的不断增加，其价格也呈下降趋势	0
可以看出，今年早熟苹果的均价较去年大幅下滑，同比降低31.66%，特别是西北地区，苹果同比去年跌幅比较大，其主要是因为2019年受前期高价影响，早熟苹果以及晚熟富士收购价格均比较高	0
所以目前的情况是，库存货源与早熟苹果之间的相互竞争，且低价的库存货不利于早熟苹果的出售，更重要的是早熟苹果由于不耐储存且销售时间短，如果不能抓住黄金销售时间，早熟果不断累库，那么将会形成早熟果后期价格下落的风险，而早熟果行情的低迷势必会对晚熟富士的开秤价造成影响	0
应季水果占领市场，苹果消费负重前行	0
而当前正处时令水果上市销售的重要时点，加上今年多数水果量多价低，所以市场对苹果消费的挤压较为严重	0
其中，鸭梨价格同比下降55.8、富士苹果价格同比下降46.7%；西瓜、香蕉、巨峰葡萄环比分别下跌31.9%、9.8%、4.7%	0
另外，由于今年降水偏多，气温也不是很高，夏季水果的消费旺季其实并为到来，叠加疫情影响，民众消费能力有所下降，苹果作为水果属于弹性消费品，所以目前的苹果需求不容乐观	0
综上所述，虽然目前苹果走货速度有所提升，但库存剩余仍处高位，需求方面，受消费能力下调影响叠加时令水果占领市场，导致苹果整体需求表现惨淡，所以，苹果市场供大于求的局面仍将持续，这也将导致苹果价格持续走低	0
短期来看，苹果盘面仍将维持低位运行，不过近期有望会出现技术回调，但整体偏弱的趋势不会改变	0
