征稿(作者：徽商期货洪敏）--2020年新季苹果期初库存量再创近年来新高，目前客商货占比较高	0
秋冬季柑橘本就对苹果有较大替代作用，今年柑橘还是丰产年，价格走低，对于苹果影响比较明显	0

柑橘价格继续下滑进一步影响苹果销售	0
秋冬季节对苹果较大的水果主要是柑橘类水果，今年柑橘产量增加，价格本就低于去年同期	0
近段时间因市场走货速度缓慢，客商采购积极性降低，产地收购价格再次下滑，对苹果销量影响比较明显	0
后期新上市货量持续增加，货源持续时间可以至次年3月份，此期间对苹果均有较大影响	0
市场其他水果价格较低，对苹果冲击较大	0
