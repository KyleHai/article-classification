河南首个苹果“保险+期货”试点项目启动	0
近日，记者从郑州商品交易所（以下简称“郑商所”）获悉，由郑商所、河南省财政厅共同支持，中原农险承保，海通期货、新湖期货、永安期货、国泰君安期货共同实施的三门峡市陕州区苹果“保险+期货”试点项目将于今年落地	0
据了解，该项目承保规模约2万亩，试点期限2年，属河南省首个苹果“保险+期货”试点项目	0
海通期货总经理助理姚弘告诉记者，“保险+期货”作为一种新型金融支农模式，实现了农业保险与期货（期权）等金融工具联动，助力解决“丰产不丰收”、价格风险难以分散等农业生产经营难题，有效帮助农民管理苹果价格波动风险，从而有利于保护农民生产积极性，推动地方特色农业稳定高质量发展，助力农业供给侧结构性改革	0
郑商所相关负责人介绍，为助力国家乡村振兴战略及区域协调发展，2016-2019年，郑商所累计支持开展了75个“保险+期货”试点项目，涉及棉花、白糖、苹果、红枣4个品种	0
其中，苹果“保险+期货”试点已开展两年，2018年，郑商所在甘肃、陕西、山西等黄土高原苹果主产区共开展9个苹果“保险+期货”分散试点项目；2019年，郑商所在陕西富县、甘肃静宁县开展2个全覆盖试点项目，惠及农户近1.6万户	0
在前期试点取得有益经验基础上，2020年，郑商所进一步创新试点建设方式，以优化试点模式和提升试点可持续性为目的，联合省财政厅在三门峡市陕州区特别开展苹果“保险+期货”试点，为探索推进试点建设新路径积累经验	0
