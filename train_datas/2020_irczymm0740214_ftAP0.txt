陕西建成全国最大矮砧苹果基地宝鸡种植面积达第一	0
<pclass='ori_titlesource'>原标题：陕西建成全国最大矮砧苹果基地宝鸡种植面积达第一	0
7月3日，全省苹果矮砧集约化栽培技术现场观摩培训会，在被专家誉为中国矮砧苹果第一县的宝鸡市凤翔县召开	0
记者从会上获悉：目前陕西已建成全国最大矮砧苹果基地，种植面积达242万亩，其中宝鸡市矮砧苹果面积占比达75%，成为全省矮砧苹果第一市	0
陕西通过苹果产业“扩矮化、减乔化”转型升级示范工程，力争由全国第一苹果大省向全国第一苹果强省转变	0
陕西前二三十年起步发展的是乔化苹果，在经历了挂果期、盛果期之后，渭北地区逐步进入树龄老化、品质下降、产量徘徊时期	0
传统乔化苹果产量低、生长期长、病虫害多、费人工等发展瓶颈长期得不到突破	0
矮砧苹果正好有效化解了这些难题，欧美发达国家已普遍采用这一新技术	0
从2013年起，陕西连续4年培训苹果矮砧栽培技术人员，邀请世界园艺果树专家授课	0
凤翔县在矮化中间砧应用上，采用短枝型品种，不仅果农易接受、收益高，还成为全国矮砧苹果栽培面积最大、效益最高的区域	0
凤翔县共栽植苹果22万亩，其中97%为矮砧栽培，占到全省的20%，目前已建成国家级苹果标准园2个，省级优质苹果示范园15个，百亩以上高标准示范园45个	0
