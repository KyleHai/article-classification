苹果：消费表现仍然一般，节后期价或回归弱势	0
<pclass='ori_titlesource'>原标题：苹果：消费表现仍然一般，节后期价或回归弱势	0
征稿(作者：方正中期期货侯芝芳）--今年以来，消费弱势为苹果市场的主旋律，期价因此不断刷新低点	0
冷库苹果高库存以及早熟苹果自上市以来弱势运行，一直压制苹果市场价格	0
9月对于苹果市场来说，原本是一个“好时节”，库存苹果经历了前期的消化，压力已经有所减轻，新季苹果尚未大量上市，西部红富士苹果开篇表现不错，再加上中秋、国庆双节提振消费市场，但是销区走货并没有跟随预期的脚步，双节对于苹果消费提振也较为一般，市场情绪再次受到打压，28日苹果近月合约再创新低，远月合约也跟随大幅回落	0
后期来看，晚熟红富士苹果集中上市在即，消费短期仍然难有大的起色，供应端的博弈或将加剧，期价预期还是以弱势为主，操作方面建议以空头思路为主	0
<divclass="img_wrapper"><imgid="0"img-size="576,327"src="http://n.sinaimg.cn/spider2020929/103/w576h327/20200929/ae5b-izwfpys5388905.jpg"style="max-width:640px"/><spanclass="img_descr">早熟苹果弱势运行	0
今年由于疫情影响，整体水果需求处于弱势环境，再加上冷库苹果的竞争，早熟苹果一直未摆脱弱势	0
卓创资讯的信息显示，早熟苹果自7月中旬上市以来，市场总体行情比较一般，最初的藤木、晨阳、秦阳等早熟品种价格开始，到后期的嘎啦、早熟富士以及红将军，整体收购价格明显低于去年	0
纸袋嘎啦主流收购价格均价为2.39元/斤，同比2019年低1.23元/斤，同比下滑幅度为33.91%，纸袋嘎啦的供应时间大概一个多月的时间，前期由于性价比不佳，价格呈现下滑态势，后期随着冷库苹果价格的回升有所上涨，不过涨幅比较有限；陕西早熟富士上市之初主流订货价格2.2-2.3元/斤，期间虽然受纸袋嘎啦价格的上涨出现上浮，但是由于整体质量一般，客商出现订单反悔的情况，后期价格一路下滑，销售尾声的时候价格相比上市初下降0.5-0.6元/斤；山东红将军上市之初的主流收购价1.8-2.4元/斤，目前进入尾期，主流销售价格为1.8-2.0元/斤，整体价格呈现稳中偏弱运行	0
库存苹果压力有一定减轻	0
目前全国苹果库存主要集中在山东地区，整体进入清库尾声	0
卓创咨讯数据显示，截至9月24日，全国苹果库存为41万吨，其中山东地区苹果库存为37万吨，在全国苹果库存中的占比为98.93%，陕西地区苹果库存为3.75万吨，在全国苹果库存中的占比为9.15%，整体库存相比正常年份高20万吨左右，对山东地区苹果市场影响较大，不过由于整体库存供应量相比前期已经大幅减少，其对市场的压力有所减轻	0
<divclass="img_wrapper"><imgid="1"img-size="523,290"src="http://n.sinaimg.cn/spider20200929/13/w523h290/20200929/4ce6-izwfpys5389706.jpg"style="max-width:640px"/><spanclass="img_descr">西部红富士苹果开篇较好	0
9月底西部地区晚熟苹果开始上市，整体收购价格呈现相对高位运行	0
目前西部苹果收购价格偏高，主要有两个方面原因，一是西部地区4月底遭遇霜冻，有一定幅度减产，二是今年陕西地区降雨充沛，无冰雹影响的地区果个普遍较大，果面质量较好，性价比较高	0
后期来看，10月迎来晚熟红富士苹果的集中供应，不过下游走货仍然表现一般的情况下，晚熟红富士苹果内部以及与其他苹果之间的博弈加剧	0
晚熟红富士苹果内部竞争主要体现为东西部地区的差异，西部地区由于没有库存苹果、客商利润尚可以及新季苹果减产质优的影响下，收购价处于相对高位，而东部地区由于存在库存苹果、客商盈利不佳以及新季苹果增产的影响下，价格预期呈现弱势，那么在销区走货没有明显好转的背景，两者预期将会向中值靠拢	0
晚熟红富士苹果与其他苹果之间的竞争主要体现为与库存苹果以及早熟苹果的竞争，晚熟红富士苹果争取市场面临价格向下靠拢的局面，不过由于库存苹果量以及早熟苹果可供应量有限，随着市场的不断消耗，其对晚熟红富士苹果拉动也将减轻，因此10月晚熟红富士苹果或是前低后高的情况	0
期货方面来看，考虑下游消费信心仍然不足，市场仍然较强利多刺激，再加上目前期货市场已经成熟，业内人士对交割品的认知以及对交割成本的控制能力有所增强，因此苹果01合约短期或延续低位区间波动	0
作者简介：侯芝芳，应用统计硕士，现任方正中期农产品研究员，主要研究品种：菜粕、苹果	0
