苹果价格的频繁波动，不利于果树的稳定种植，而2018年上半年，苹果期货价格上涨幅度超过70%	0
2018年，天气异常，果树受灾严重，根据不同机构的调研结果，苹果减产24%—56%，市场出现供应紧张局面	0
受减产的提振，苹果期货价格于2018年11月16日攀升到最高点12626元/吨	0
甘肃是我国苹果主产区之一，果树面积居全国第二，产量居全国第四	0
甘肃省天水市秦安县属陇中黄土高原西部梁峁沟壑区，该县苹果种植区域包含兴国镇、西川镇、莲花镇等十七个乡镇，果树种植面积达60万亩，挂果面积约36万亩，产量在50万—75万吨，果农约10.2万户，苹果收入为其支柱收入	0
近几年，苹果价格大起大落，尽管市场提供了具有价格发现、风险管理功能的苹果期货工具，但由于果农缺乏金融知识和运用技能，收入仍然起伏不定，对全县脱贫攻坚造成一定阻碍	0
据统计，截至2018年7月，全县共有17镇428村10.2万户39.6万人受灾，农作物受灾面积6.1万公顷，其中苹果树受灾3.7万公顷，成灾面积3.4万公顷，其中苹果树成灾2万公顷，造成直接经济损失约18.6亿元	0
2018年是秦安县苹果产业发展转型的重要时期，自然灾害和价格波动等多重因素的困扰导致群众对市场前景信心不足，对现代产业新技术、新业态、新模式投入和参与的积极性不高，产业发展遭遇瓶颈	0
考虑到秦安县脱贫攻坚的实际情况，以种植果树为生的贫困户对国家政策及苹果价格有很强的敏感度	0
对于他们来说，一旦苹果价格大幅下跌，基本的生活就难以保证了	0
本次试点项目的保险及场外期权挂钩标的均为项目承做期间郑商所苹果期货主力1905合约	0
2018年，在郑商所支持下，宏源期货联合安信农保和太保财险在秦安县开展苹果“保险+期货”试点项目，主要流程是：	0
保险公司推出苹果目标价格保险产品，并向果农销售	0
按照合同约定，以保险初始日至索赔日，即2018年11月20日—2019年1月10日期间各交易日的苹果期货1905合约收盘价的算术平均值11503.14元/吨作为理赔依据，[（目标价格12300元/吨-期货合约平均收盘价11503.14元/吨）×下行参与率0.85+目标价格12300元/吨×0.5%]=理赔金额738.83元/吨	0
政策方面，甘肃省发布《甘肃省2018—2020年农业保险助推脱贫攻坚实施方案》，天水市发布《关于支持苹果“保险+期货”项目的承诺函》，在政策上予以支持	0
借鉴本次试点项目的成功经验，秦安县于2019年3月27日再次组织实施两万吨苹果的“保险+期货”项目，进一步助力该县脱贫攻坚，为“保险+期货”县域全覆盖打下基础	0
秦安县作为农业农村部划定的西北黄土高原苹果优势产区，近年来始终把苹果产业作为全县经济发展的重中之重	0
在“保险+期货”试点项目的带动下，该县还设立了苹果期货交割仓库，利用现代金融工具为产业发展保驾护航	0
本次试点项目保险产品对应的期权产品为保护型亚美式期权，特点一是赔付概率为100%，具体是苹果价格下跌时，按下跌价差赔付果农，而苹果价格上涨时，给予一定补偿，作用是帮助果农锁定最低销售价格；特点二是保障与费用均衡，基于大数据统计特征，将参与率和价差区间适当调整，降低项目成本并大概率不影响赔付效果；特点三是灵活性高，项目期间内果农均有权提前实现赔付；特点四是保险通道费下降至7%	0
基本上，参保果农较未参保果农每斤多卖3角7分钱，这在很大程度上解决了苹果种植户最关心的价格风险问题	0
另外，尽管通过“保险+期货”防范和规避市场风险是一项极为复杂的经济活动，但对于果农来说只是简单购买了苹果价格保险，后续的风险化解过程全部交由专业的金融机构来完成	0
参保果农都对该试点给予积极评价，莲花镇袁山村张姓参保果农表示，自己全家的收入全靠几亩苹果树，一年到头，从种植、修剪到套袋、摘袋，小心翼翼地防病治虫，可一旦碰到伤冻，辛辛苦苦收获的苹果也卖不了几个钱，一度都想改种别的品种，或者干脆去城市打工，而通过苹果“保险+期货”试点项目，心里踏实了，能够安心搞生产了	0
