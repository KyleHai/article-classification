柑橘大量上市苹果下跌预期兑现	0
今年遇到的情况主要是由于春节提前和各品种因为天气原因集中成熟，从而导致了扎堆上市，柑橘自身的扎堆上市不仅给自身的价格带来压力，也给苹果的销售价格带来很大压力	0
2018年3-4月份，苹果因为开花期遭遇了几十年不遇的倒春寒，使得苹果大面积减产，苹果现货和期货也出现了罕见的历史性上涨行情，2019年苹果产量回顾到正常水平	0
但苹果的价格会什么时候见底呢	0
按照往年的正常的水平，以及历史上出现的极端低价参考可以看出，苹果的现货底部价位区间在2-6元/公斤，2019年的产量虽然有所恢复，但总产量依然低于2017年，因此出现极端下跌情况的概率并没有那么大	0
观点小结：现在一直持续到3月份，柑橘大量的上市会一直压制着水果市场的价格，但一季度结束后，压制作用会逐渐减小，而且届时到了苹果的开花期，市场可能会出现新的情况，因此我们预计苹果期货端一季度或将暂维持偏空格局	0
