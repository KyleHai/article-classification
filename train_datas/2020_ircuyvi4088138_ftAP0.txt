西北农林科技大学白水苹果试验站入选国家现代农业科技示范展示基地	0
<pclass='ori_titlesource'>原标题：西北农林科技大学白水苹果试验站入选国家现代农业科技示范展示基地	0
近日，农业农村部公布了国家现代农业科技示范展示基地名单，西北农林科技大学赵政阳教授牵头申报的白水苹果试验站入选国家现代农业科技示范展示基地	0
国家现代农业科技示范展示基地依托西北农林科技大学白水苹果试验站，试验站占地163亩，总投资2000余万元，是该校自主建立的拥有土地产权的国内首个专业性苹果试验示范站	0
试验示范站围绕苹果新品种、新技术研究，组建了以赵政阳教授为组长，苹果育种、栽培、植保、土肥、贮藏等29名科教人员为团队的推广队伍，并吸纳地方科技人员，积极探索出了“大学+试验示范站(基地)+示范户”的科技入户通道，构建起产、学、研三位一体的试验示范站和教学实习基地	0
经过十多年的努力，苹果试验示范站在基础设施建设、科学研究、科技示范、科技培训、人才培养等方面取得了初步进展和成效	0
苹果试验示范站的建立，实现了农业教育、科研与技术推广相结合；大学优势学科，核心技术与产业发展相结合；地方政府、大学与企业、农业组织相结合；大学科技专家与地方科技人员、果农相结合	0
同时，以试验示范站为中心，通过技术培训和示范推广，辐射带动了白水县及周边苹果产业的全面进步和发展，受到了当地政府和群众的充分肯定，并在国内外产生影响	0
苹果试验示范站兼具科学研究、技术推广、人才培养三大功能，并朝着把试验示范站建设成为产学研于一体，国内一流、国际知名的高水平的专业性试验示范站建设目标而努力	0
