二、从供应端分析其他水果对苹果的替代效应	0
供应端分析其他水果对苹果的替代效应主要考虑一下两个因素，一是该类水果的供给量，二是该类水果的上市时间	0
苹果的上市时间是从7月底开始到11月,其中6月至8月为早中熟苹果集中上市时间，晚熟苹果由于储存技术和冷库建设，全年都有销售	0
在六月份之前，夏季水果还未上市，市场上的应季水果主要是菠萝、梨、苹果、樱桃、草莓等，考虑到菠萝、草莓、樱桃等水果的产量不高，加上此类水果不易保存、价格偏高等因素，对苹果的替代效应较弱	0
此时这些水果的大量上市对苹果的需求产生巨大的冲击	0
8月份以后，早熟的苹果开始上市，数量有限且价格相对较低，各夏季水果也逐渐从市场中淡去	0
此时苹果面对的最大的冲击是10月上市的桔子、橙子和柚子等	0
从图4可以看出，往年的苹果产量都大于柑桔产量，但是18年开始柑桔的产量超过了苹果的产量	0
2019年苹果产量的增幅是8.13%，柑桔的产量增幅是10.78%，柑橘在短时间内集中上市对苹果的冲击非常大，特别是体现在年货储备上，蜜桔、砂糖桔、贡柑、沃柑等都在同苹果抢夺年货市场	0
从图5-9可大致看出，苹果与鸭梨、香蕉的出货量具有同增同减的趋势，苹果与葡萄、菠萝、西瓜反向变动的趋势更为明显	0
细一点看，苹果批发量的高峰一般每年的1月份，低谷在7、8月附近，这可能与苹果的库存量和消费需求有所关联	0
从另一种角度，这些夏季上市的水果填补了苹果夏季批发不足的空缺	0
三、从需求端分析其他水果对苹果的替代效应	0
从需求端分析其他水果对苹果的替代效应主要有三个方面的因素，一是人们的消费需求引起的替代效应，我们称之为拉力；另一个是价格因素导致人们需求的转移，我们称之为推力；最后一个因素是收入，收入的变化对上述两者因素都会产生影响	0
据果品营养之类的文章称，人们对苹果的营养需求可能来自“anappleaday，keepthedoctoraway”，从而苹果就成了健康果的代名词，并有了相对硬性的需求，一天一个苹果，减肥人士也深信苹果具有减肥的功效	0
其他的水果健康功效大致类似，不过都没苹果那么出名，而且会存在含糖量高、偏寒或偏热等缺点，苹果的品性最为优越，人们处于健康而对苹果有所偏好是情有可原的	0
从自身的满足程度上看，苹果基本四季皆宜，不过夏天需求会有所降低，因为苹果水分较少，夏天喜多汁水果；其余季节，尤其冬季、过年，其他水果也不多，天气寒冷，对水果需求也不大，此时对苹果的需求就上来了	0
从万得的数据来看，西瓜、鸭梨、香蕉的价格普遍低于苹果，从这个角度上说，这些水果对苹果都有替代的推力	0
柑桔方面的价格大致也是低于苹果，而葡萄、草莓等产量较小的，价格高于苹果，对它们的需求应该只是偶尔零食级	0
从四者价格波动的相关性来看，苹果与周期性较强的西瓜的相关性较弱，与鸭梨的相关性最强为0.38，与香蕉的相关性为0.13.由此推知，水果价格基本上都是正相关，齐涨齐跌	0
从下图的走势来看，居民消费水平直线上升，收入的增加导致消费能力的提升，从而增加正常商品的需求，减少对低档品的需求苹果消费量波动向上，可见其斜率小于居民消费水平的斜率；当然，收入的增加可能对高档苹果和低档苹果的影响不一样，所以结合苹果的进口数据进行分析	0
可以看出进口量在10年内有增有减，进口减少的年份是国内苹果的大年，而增加年份斜率明显大于苹果消费量的斜率，甚至超过居民消费水平	0
从变动率角度，人均消费水平从2010年的10550元增加至2019年27563元，增幅161.2%，国内苹果消费10年增幅为52.88%，苹果进口量增幅为55.40%	0
