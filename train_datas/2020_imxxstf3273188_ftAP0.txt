银河证券推荐扶贫项目“静宁苹果”在新三板正式挂牌	0
上证报中国证券讯网2月20日，由银河证券推荐的平凉静宁苹果产业发展股份有限公司（下称“静宁苹果”）在新三板正式挂牌	0
甘肃静宁苹果产业作为静宁县支柱性产业，对静宁县的脱贫攻坚工作具有重要意义	0
“静宁苹果”品牌知名度较高，成为银河证券推荐的第一个新三板扶贫项目	0
近几年来，银河证券在静宁县共投入扶贫资金超过1亿元，帮助建设2万亩生态林、银河希望小学、苹果期货交割库等项目，派驻4名挂职干部，消费扶贫累计1600余万元	0
去年9月，由银河证券重点参与的甘肃省唯一的苹果“保险+期货”县域全覆盖项目在静宁启动，标志着国内迄今为止最大一单苹果“保险+期货”项目正式落地	0
这对支持静宁县苹果产业发展、促进金融工具与农业生产深度融合、助力静宁县脱贫摘帽工作提供了有力保障，也为金融扶贫探索了一条有效的新路	0
下一步，银河证券将以“静宁苹果”新三板挂牌为契机，进一步提升贫困地区相关主体利用多层次资本市场的资源、工具及市场化机制服务本地区脱贫致富，助力打赢脱贫攻坚战	0
