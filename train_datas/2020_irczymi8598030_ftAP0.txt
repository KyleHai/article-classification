午评：苹果主力及远月合约全涨停沪锡涨近5%	0
4月27日，国内期市早盘分化明显，苹果主力及远月合约全涨停，其他农产品则多数下跌，白糖跌逾3%，棕榈油跌近3%；能化品涨跌不一，lpg涨逾6%，燃油跌逾6%，原油跌逾4%；基本金属全线上涨，沪锡涨近5%，沪镍、沪锌涨逾2%	0
苹果主产区花期冻灾再现	0
很多人都吃过苹果，但是见过苹果花的人却比较少	0
苹果开花期因各地气候不同而有很大差异，一般在4~5月份，每花芽有花3~7朵，且中心花先开	0
苹果是异花授粉植物，大部分品种自花不能结实	0
一德期货在研究报告中指出，产区的冻灾新闻引发了市场对于新季苹果看涨心态	0
由于2018年倒春寒曾导致苹果大量减产，因此4~5月的天气问题备受市场关注	0
从3月下旬到目前为止，西北苹果主产区已经经历了三波春寒	0
有记者从陕西延安一苹果种植大户的微信群提供的现场视频发现，霜冻降温天气过后，苹果园里变成了另外一番模样，不少花朵已经凋谢了，只有少量的粉白色的苹果花还挂在枝头	0
对很多果农来说，低温霜冻对于花期的苹果影响很大，遭遇霜冻的果农苦不堪言	0
那么，对消费者而言，今年吃的苹果价格会涨价吗	0
记者从该种植大户的微信群反馈的情况来看，目前现货市场的苹果已经涨价2毛钱	0
私募排排网研究中心告诉记者，我们认为苹果价格大概率不会涨价，一方面是此次霜冻属于区域性的，并不是全国性的，现在种苹果的地区越来越多；另一方面，现在苹果才逐渐进入盛花期和坐果期，霜冻预警对苹果坐果率影响相对较小	0
瑞达期货认为，受到产区持续降温，市场担心冻害对苹果花产生不利影响，加之五一备货需求及资金助力，国内苹果现货价格趋于稳定，目前产区库存较之前继续减少，部分地区剩余果农货源，果农惜售，挺价心理较强，加之受前期收购价格影响，现货价格较为坚挺	0
此外，五一节日备货需求持续，均给予苹果一定的价格支撑	0
近期主产区多有霜冻预警，且陕西地区苹果树仍处于花期，仍需4~5天左右坐果，河北多地出现冻害，或在一定程度上提振苹果市场	0
整体上，预计五一之前，苹果价格或有所走强	0
