今日，苹果主力合约2005跌破期货上市以来的05合约最低点，从技术面来讲，上下价格极点的破位一般都有一定的启示作用	0
但技术面的背后隐藏的是基本面的变化，现在苹果破掉历史6471元/吨低位，说明此刻的基本面已经比历史低点时还要差	0
让我们了解一下苹果本周的基本面动态吧	0
现货方面，栖霞产区苹果报价，纸袋富士80#以上：2.50-2.80元/斤（一二级）；纸袋富士75#：2.00-2.20元/斤（一二级）；纸袋富士80#以上：1.80-2.00元/斤（统货）	0
库存方面，目前苹果冷库库存正处于近五年高位，19/20冷库剩余量较正常年份17年同期多出150万吨，明显高于前年同期	0
统计数据显示2020年2月初冷库苹果剩余量约为895.7万吨，冷库剩余量同比2019年多111.09%	0
近五年2月初全国冷库苹果余量平均值为668.9万吨，2020年比均值高226.8万吨，高出均值33.9%，明显高于其他年份	0
消费方面，由于2019年丰产，过年早、库存较大又遇到肺炎疫情，苹果的积压使得果农当前很少有观望惜售的情况，大多有序处理库存，顺价销售，比前几年明显理智很多，这也使得年后行情维持顺畅的局面	0
而江南、华南大部地区最近阴雨天气较多，不利于批发市场交易，南方地区气温普遍比常年偏高2-3度，苹果货架期的销售质量面临较大考验	0
观点小结：当前的苹果的基本面仍然处于偏弱的状态，但随着疫情的好转，物流的限制放开，产区冷库也开集中出库，值得注意的是产区现在行情较旺走货较快，但仍以低价货源为主，而销区短期内仍保持较大压力，清明节之前市场上的苹果供应量很大，当前苹果期货端价格已经打破了历史低位，后续或继续以震荡走低为主，行情只能走一步看一步，等待清明节前后的终端消费市场表现；另外，也要时刻关注天气方面的变化，苹果开花期是比较重要的阶段	0
