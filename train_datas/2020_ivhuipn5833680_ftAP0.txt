日前，郑商所举办2019/2020年度“保险+期货”结项评审会，包括白糖、苹果、红枣3个品种在内的共计9个项目参与评审	0
根据对项目实施效果、项目运作成本、项目产生的社会效应等方面的评审，最终云南孟连及景谷白糖项目、陕西富县苹果项目、新疆阿拉尔红枣项目被评为郑商所2019/2020年度“保险+期货”优秀试点项目	0
郑商所相关负责人表示，2019年是郑商所“保险+期货”试点建设“由点及面”迈上新台阶的关键一年，郑商所以扩面工作为重点，投入约4500万元在白糖、苹果品种上各支持开展了2个全覆盖试点项目，同时将红枣品种纳入试点范围，支持开展了5个小规模分散试点项目	0
苹果产业是陕西富县的支柱产业，种植苹果单项收入占当地果农家庭总收入70%以上	0
在2019年苹果产业增产、价格走低的形势下，“保险+期货”试点有效降低了当地果农增产不增收的风险	0
2019年11月，由太平洋财险承保，浙商期货、方正中期期货、锦泰期货合作实施的富县苹果“保险+期货”试点项目率先进入理赔阶段，1780户果农获得1800万元赔付，赔付率达到131%	0
试点成效得到当地政府的肯定与果农的认可，“种苹果要跟着期货行情走”成为当地一些果农的经验之谈	0
据郑商所相关负责人介绍，2016—2019年，郑商所累计支持开展“保险+期货”试点项目75个，涉及棉花、白糖、苹果、红枣4个品种，覆盖新疆、甘肃、广西等11个省（自治区），涉及农户10万余户，累计赔付金额超过1亿元，平均赔付率近70%	0
计划主导开展试点项目31个，其中“保险+期货”试点项目21个，覆盖白糖、苹果、红枣三个品种；“农民合作社+场外期权”试点项目10个，覆盖苹果、红枣两个品种	0
此外，郑商所还计划联合河南省财政厅在三门峡市开展苹果“保险+期货”试点	0
