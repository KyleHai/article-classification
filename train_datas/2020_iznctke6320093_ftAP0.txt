12月8日至9日，郑商所总经理鲁东升一行赴兰州与甘肃省人民政府签订战略合作协议，并调研中央财政农业保险奖补试点覆盖苹果“保险+期货”项目开展情况和马铃薯产业发展现状	0
甘肃省地处黄土高原、青藏高原和内蒙古高原交汇地带，山脉纵横交错，海拔相差悬殊，正是这种山地型高原地貌的气候环境使得甘肃成为中国“最甜”苹果的产区之一，甘肃多地都是国家苹果的优势产区，苹果产业成为当地的支柱产业，如何避免价格波动对产业带来的风险成为甘肃苹果产业面临的重要课题之一	0
苹果期货急果农忧“保险+期货”解行业困	0
“一个一个的小苹果是我们的‘致富果’，承载了我们果农的希望	0
”甘肃通渭县县长谢占武在交流中说，“上世纪八十年代前‘面朝黄土背朝天，十年九旱闹饥荒’的情景不复存在了，农民靠这些苹果脱了贫	0
在中国证监会的指导下，郑商所于2016年开始苹果期货的研究	0
据期货日报记者了解，面临苹果产业标准化程度相对较低、国际期货市场上无鲜果期货经验可借鉴等困难，郑商所按照既方便市场主体参与又严格保障市场安全运行的原则，完成了苹果期货的研发设计，并于2017年12月22日挂牌上市，苹果期货成为全球首个鲜果类期货品种	0
经过两年多的运行，苹果期货在服务产业、助力脱贫攻坚方面的作用逐步显现，期货业与保险业优势互补的价格保险产品苹果“保险+期货”项目试点也应运而生	0
2018年苹果期货上市初期，郑商所就将甘肃纳入苹果“保险+期货”试点范围，覆盖当地8个苹果主产县	0
2018年以来，郑商所在甘肃共开展试点项目12个，累计承保苹果现货12.7万吨覆盖8.5万亩苹果地，总保障金额近10亿元，惠及农户超过2万户，其中1.4万户（占比73%）为贫困户，除2020年尚未结束的6个项目外，已完成项目产生赔付2579.52万元，户均赔付2150元，平均赔付率达82%，参与项目的静宁县、秦安县、麦积区等均为国家级贫困县，自项目实施以来为当地苹果种植户提供了坚实的价格风险保障，助力脱贫攻坚效果显著	0
“苹果‘保险+期货’项目的成功试点，让参保受益的贫困农户大受鼓舞	0
”秦安县副县长李虎介绍，“保险+期货”项目的开展进一步提高了苹果产业抵御市场风险的能力，完善了产业发展的保障服务体系，探索出了一条金融扶持产业发展、助推脱贫攻坚的新路子，不仅得到了广大农户的充分肯定，同时也得到了甘肃省委领导的高度赞扬	0
”静宁县副县长杜晓光告诉期货日报记者，2019年和2020年，郑商所联合中国投资有限责任公司连续支持静宁县实施苹果“保险+期货”县域全覆盖项目	0
2019年为全县24乡镇8713户贫困户投保苹果4.6万吨，实现了种植苹果未脱贫户全覆盖，项目赔付1676.34万元、赔付率72.88%，贫困户每户获赔1676.34元；2020年为2140户“三类户”（2019年年底未脱贫户、监测户和边缘户）和3个贫困村、10家合作社的3763户果农投保4.8万吨，在保费资金来源方面实现农户自缴保费356万元，农户主动参保意识显著提高	0
人保财险甘肃省分公司党委书记董彦明告诉记者，2020年12月8日，通渭县农业农村局主动联系人保财险、中信期货、永安期货共同签署苹果“保险+期货”合作协议	0
“郑商所上市苹果期货并实施‘保险+期货’试点，为当地苹果种植户提供了坚实的价格风险保障，对稳定果农和销售企业生产经营起到很大积极作用	0
2019年，中央财政首次在全国开展地方特色农产品保险奖补试点，甘肃省争取到苹果中央奖补试点10万亩，2019年至2020年，甘肃省农业农村厅、甘肃省财政厅《关于做好中央财政对苹果和中药材保险奖补试点工作的通知》《关于扩大中央财政对地方优势特色农产品保险以奖代补试点范围的通知》中明确将苹果“保险+期货”纳入甘肃省苹果保险中央财政奖补试点，项目实施区域为设有苹果交割库的天水市麦积区、天水市秦安县、平凉市静宁县、庆阳市西峰区，通过‘保险+期货’的模式累计赔付金额近4000万，惠及农户超过3.7万户	0
2020年，甘肃省扩大苹果“保险+期货”中央奖补试点覆盖范围至15万亩，累计承保苹果12.3万亩，项目覆盖农户2.95万户，其中贫困户为2.3万户	0
甘肃省将苹果“保险+期货”纳入中央财政奖补试点，实现了以财政资金为主推动“保险+期货”项目落地实施的新模式，在财政资金常态化、机制化支持“保险+期货”方面率先取得突破，这也是完善农业支持保护政策方面的重要探索	0
鲁东升表示，甘肃苹果“保险+期货”项目实施效果明显，果农收入得到保障，苹果产业稳定发展，在甘肃脱贫攻坚方面发挥了重要作用，也必将在助力甘肃乡村振兴方面发挥积极作用	0
曾在甘肃农科院工作的莱柯土豆总经理李进福表示，与苹果产业存在红富士这一主流品种不同，马铃薯品种较多，v7、青薯9号、黄金薯、中加2号、荷兰15等众多品种都有市场需求，按照品种划分将单一品种确定为交割标的难度较大，需要对马铃薯主产区的主流品种进行详细调研才有可能设计好交割标准，但是工作量巨大	0
鲁东升表示，马铃薯期货研发意义重大，甘肃省、定西市政府和马铃薯产业相关企业、农户热切期盼推出马铃薯期货，郑商所将提高政治站位，将马铃薯期货研发作为后期品种研发工作的重中之重，深入总结同为生鲜品种的苹果期货研发和监管经验，努力化解品种繁多、主流品种更替速度快、质检机构缺失等产业现状带来的期货合约规则设计难题，全力推进马铃薯期货研发，扩大期货市场服务甘肃发展的覆盖范围，为进一步助力巩固甘肃脱贫攻坚成果，助推甘肃落实乡村振兴战略，服务国家粮食安全战略贡献期货力量	0
