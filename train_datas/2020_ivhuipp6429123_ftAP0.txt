<pclass='ori_titlesource'>原标题：2020年9月18日，期货日报：“保险+期货”护航延安果农收获“金苹果”来源：新闻信息部易盛	0
近日，在陕西省延安市落地两个苹果全覆盖“保险+期货”项目分别在富县和延长县启动	0
据悉，延安市是世界最佳苹果优生区，年平均气温、年降水量等7项气象指标全部符合世界优质苹果生长条件	0
全市13个县（区）全部为省级优质苹果生产基地县，是世界上集中连片种植苹果面积最大的区域	0
改革开放以来，延安依托资源优势，坚定不移走以苹果为主的特色农业产业富民之路，形成的“小苹果、大产业、大带动”延安产业扶贫模式在全国推广	0
期货日报记者了解到，由太平洋财产保险陕西分公司、浙商期货、方正中期期货、国投安信期货共同实施的富县的苹果全覆盖“保险+期货”项目参保种植面积2.3万亩，可保障苹果现货约3.46万吨	0
项目采取“贫困户优先，一般户自愿”的原则，富县从事苹果种植的建档立卡贫困户可优先投保，争取做到贫困户2000余户全覆盖	0
其中不少贫困户代表2019年就参与过了富县的苹果“保险+期货”项目，贫困户代表均表示，去年项目成效很好，今年也要积极参与	0
延长县苹果全覆盖“保险+期货”项目是由中国人民财产保险陕西省分公司、长江期货、中信期货、东证期货联合实施，项目参保种植面积达到2万亩，可保障苹果现货约3万吨，预计承保苹果种植户2000余户	0
2017年以来，延长县通过苹果“保险+期货”金融扶贫模式，进一步增强了果农应对市场风险的能力，保障其基本收益，有效地巩固了脱贫攻坚成效	0
